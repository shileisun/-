## 一、赚钱的本质认知

### 1.1 三大财务报表的"经营者视角"

| 报表 | 官方定义 | 经营者要记住的一句话 |
|------|---------|---------------------|
| **利润表** | 反映某一期间经营成果（亏损或盈利） | 我这一年到底赚没赚钱？ |
| **资产负债表** | 反映某时点财务状况，拥有多少财产，承担多少债务 | 我的家底到底有多厚？ |
| **现金流量表** | 反映某期间现金收支情况 | 钱到底去哪了？ |

### 1.2 必须打破的5大认知误区

| 误区 | 真相 |
|------|------|
| 规模大 = 实力强 | 规模是有质量的——利润 |
| 有利润 = 不缺钱 | 利润是有质量的——现金流 |
| 融资越多越好 | 融资带来财务风险（利息负担） |
| 企业资产 = 老板身价 | 别忘了负债 |
| 企业的钱 = 老板的钱 | 法律主体不同，随意挪用=违法 |

### 1.3 权责发生制（经营者的"第一课"）

**核心原则**：成本费用在受益期间内平均分配，不是付钱的时候算。

| 概念 | 案例 | 为什么重要 |
|------|------|-----------|
| **折旧** | 服务器10万/5年 → 月折旧1667元 | 资产成本要分期消化，不能一次吃掉利润 |
| **计提** | 薪酬下月发 → 本月先计入成本 | 真实反映当月经营成本 |
| **摊销** | 房租押一付三11万 → 月摊销3万 | 预付费用按月均摊，不做"假盈利" |

### 1.4 收入 ≠ 收款（最容易被搞混的4个概念）

```
收入 ≠ 收款  ≠  收款÷(1+税率)  ≠  开票金额  ≠  开票金额÷(1+税率)
```

**SaaS收入确认规则**：

| 收入类型 | 确认规则 | 举例 |
|----------|---------|------|
| 订阅 | 开通期间内按月分摊 | 12万/年合同 → 月收入1万 |
| 实施 | 投入法，按投入进度确认 | 50人天项目，完成20天 → 确认40%收入 |
| 客开 | 客户生命周期内分摊 | 3年合同 → 分3年确认 |
| 对接 | 验收时一次性确认 | 验收通过当天确认 |

**收入不含税**：合同总收入 = 合同金额 ÷ (1+税率)，SaaS服务税率6%。

### 1.5 利润 ≠ 现金收入 - 现金支出

**利润表的关键结构**：

```
营业收入（不含税）
- 营业成本
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 其他收益
- 各项减值/损失
= 营业利润
+ 营业外收入 - 营业外支出
= 利润总额
- 所得税费用
= 净利润
```

**核心提醒**：看利润表不光看"净利润"，还要看"营业利润"（主营赚不赚钱），看"毛利"（产品本身有没有竞争力）。

## 二、算大账：代理商长期财务预算模型

### 2.1 经营目标参数设定

```
【关键参数清单】
- 新购ARR年增长率
- 新购年限（合同平均年限）
- 增购率（增购ARR ÷ 应续ARR）
- 续费率（分A级/C级客户）
- 续费年限
- 一次性占前端比（实施费/合同总额）
```

### 2.2 产出三层递进模型

```
ARR规模 → 合同收款 → 返款收入
  │          │          │
期初ARR     新购      新购+增购返款(**%)
本年应续    增购      一次性返款(**%)
新购ARR     一次性    续费返款(**%)
增购ARR     续费
续费ARR     前端合计
流失ARR     整体合计
期末ARR
```

**关键公式**：

| 指标 | 公式 |
|------|------|
| 期末ARR | 期初ARR + 新购ARR + 增购ARR - 流失ARR |
| 应续ARR | 以前年度ARR在本年应续 + 当年新增ARR在本年应续 |
| 续费ARR | 应续ARR × 续费率 |
| 流失ARR | 应续ARR × (1-续费率) |
| 前端合同收款 | 新购 + 增购 + 一次性 |

### 2.3 投入结构（3类人员 + 3类费用）

#### 人员费用

| 角色 | HC计算依据 | 薪酬构成 |
|------|-----------|---------|
| **销售** | 人均收款XX万 → HC = 目标收款 ÷ 人均 | 年薪包 + 产品提成(%返款) + 实施提成(%返款) |
| **售前/交付** | 人均实施XX万 → HC = 目标实施 ÷ 人均 | 年薪包 + 提成(%前端收款) |
| **CSM** | 人均续费XX万 → HC = 目标续费 ÷ 人均 | 年薪包 + 提成(%续费收款) |

#### 费用类别

| 类别 | 说明 | 典型占比 |
|------|------|---------|
| 销售费用 | 招待/差旅等 | 按前端收款%估算 |
| 固定费用 | 房租物业 | 按人均每月XX元 |
| 税费 | 增值税 | 按返款金额XX% |
| 其他 | 杂项 | 按返款金额XX% |

### 2.4 利润拆解公式

```
产品利润 = 产品返款 - 产品提成
+ 交付利润 = 交付返款 - 交付人工 - 交付提成
+ 续费利润 = 续费返款 - CSM人工 - CSM提成
- 公共费用 = 固定费用 + 销售费用 + 其他
- 增值税
= 现金利润

利润率 = 现金利润 ÷ 返款总收入
```

### 2.5 人效指标

| 维度 | 指标 |
|------|------|
| HC合计 | 销售 + 交付 + CSM + 职能 |
| 人均收款（客户收款维度） | 整体收款 ÷ HC合计 |

## 三、算小账：项目级利润

### 3.1 人天成本单价公式

```
人天成本单价 = 人均人天成本 ÷ （年工作日 × 有效利用率）

例：人均年薪20万，年工作日250天，有效利用率80%
→ 人天成本单价 = 200000 ÷ (250 × 0.8) = 1000元/天
```

### 3.2 项目利润速算案例

> **场景**：产品4.36万(2折/100账号/1年) + 实施50人天×1800=9万，拟报价13万
> 客户预算10万。费用合计9.65万。
>
> 10万 - 9.65万 - 税0.56万 = **亏损2100元**
>
> ⚠️ **教训**：不算细账就报价，看似13万的项目，客户砍到10万就亏了。

### 3.3 项目利润检查清单

1. 产品费能不能覆盖提成？
2. 实施人天报价能不能覆盖实际成本？
3. 异地项目的差旅/住宿谁承担？
4. 税率6%是否已计入？
5. 是否有隐性实施投入（培训、客开）？

## 四、SaaS经营逻辑

### 4.1 代理商盈利模型 vs 厂商盈利模型

| 维度 | SaaS厂商 | SaaS代理商 |
|------|---------|-----------|
| 融资 | ✗（资本寒冬） | ✓（轻资产） |
| IPO | ✗（门槛提高） | ✓（现金流好） |
| 成本优势 | ✓（规模效应） | ✗（人力密集） |
| 核心逻辑 | 新客不亏损，续费有利润 | 新客有利润，续费养团队 |
| 导向转变 | 从增长导向→利润导向 | **以利润为导向，以规模为牵引** |

### 4.2 代理商价值链模型

```
市场开源 → 方案与销售 → 交付 → CSM
   │           │          │      │
开源策略    销售策略    交付盈利  持续经营
开源能力    方案能力    项目管理  续费率
市场投入    损益测算    交付方法论 增购策略
           正确客户     项目利润
           高质量合同   项目应收
```

### 4.3 ARR增长陷阱

**核心公式**：

```
新购率 + 增购率 > 流失率
（新购率 = 新购ARR ÷ 应续ARR）
```

**NGR vs NDR 十年模拟**：

| NGR | NDR | Y1→Y10增长 |
|-----|-----|-----------|
| 0% | 80% | 几乎没有增长（ARR规模始终≈3倍Y1） |
| 5% | 95% | 8倍增长 |
| 10% | 100% | 12倍增长 |
| 15% | 105% | 18倍增长 |
| 20% | 110% | 28倍增长 |
| 25% | 115% | 41倍增长 |
| 30% | 120% | 59倍增长 |

**惨痛教训**：随着ARR规模增大，即便保持90%续费率，流失ARR的绝对值也在增长。不加大新购/增购投入，ARR就会停滞甚至倒退——这就是"ARR增长陷阱"。

### 4.4 订阅 vs 实施的返款对比

| 方案 | Y1返款 | 5年累计返款 | 结论 |
|------|--------|-----------|------|
| 产品10万+实施10万 | 16万 | 32万 | 订阅更高 |
| 产品5万+实施15万 | 18万 | 26万 | 短期高，长期低 |

**结论**：在持续续费的前提下，订阅收益更高。每1元实施费相当于"本金"，次年通过续费收回，后续每年相当于40%的收益率。

### 4.5 客户生命周期价值

| 续费率 | 生命周期(年) |
|--------|------------|
| 80% | 5年 |
| 85% | 7年 |
| 90% | 10年 |
| 95% | 20年 |

```
客户生命周期 = 1 ÷ 流失率
例：续费率90% → 流失率10% → 生命周期 = 1÷0.1 = 10年
```

### 4.6 加大新客户投入 vs 加大老客户投入

**经营分析决策框架**：

```
问题：续费能覆盖团队成本时，应该把钱投给新客户开拓还是老客户经营？

判断维度：
1. 获客成本 vs 增购成本（通常增购成本远低于获客）
2. NDR现状（NDR>100%说明老客户在增长）
3. 销售团队产能（有没有余力开拓新客户）
4. 客户甜区饱和度（目标市场吃完了吗）
```

## 五、高质量签约

### 5.1 做正确客户的三个原则

1. **以商业利益为前提**：以客户为中心 ≠ 客户要什么都给
2. **做愿意为价值付费的客户**：不降价抢单，不跟乞丐客户纠缠
3. **做甜区客户**：不求大，做自己能服务好的体量

### 5.2 高质量合同三标准

| 标准 | 量化指标 | 为什么 |
|------|---------|--------|
| **有利润** | 产品返款可覆盖提成+分摊 | 没利润的合同是"负债" |
| **能耗低** | 实施 ÷ ARR ≤ 2（理想1.5~2） | 实施占比太高说明交付重，长期不划算 |
| **ARR占比高** | 订阅ARR ÷ 合同总额 ≥ 60% | 续费收入才是长期价值 |

### 5.3 合同质量速判

| 情形 | 判断 |
|------|------|
| 产品10万 + 实施20万 | ⚠️ 实施是产品的2倍，能耗太高 |
| 产品15万 + 实施8万 | ✅ 产品为主，实施为辅 |
| 产品5万(1年) + 实施5万 | ⚠️ 一年合同，实施占比50%，续费不确定 |

## 六、成本管控（降本）

### 6.1 七大易被轻视的成本

| # | 成本项 | 特性 | 典型占比/金额 |
|---|--------|------|-------------|
| 01 | 人工成本的刚性 | 隐性、不易理解、弹性 | 薪酬+公积金+奖金+房租分摊 |
| 02 | 续费成本 | 常被低估 | 实施费的20% |
| 03 | 服务器/运维成本 | 逐年增长 | - |
| 04 | 增值税 | 合同额的4.5%（法定刚性） | 6%税率 × 75%增值部分 |
| 05 | 房租物业/职能管理 | 刚性成本 | 损益模型约20%公摊 |
| 06 | 差旅费 | 隐性、弹性 | 客户承担/提供食宿/双人同住 |
| 07 | 第三产品成本 | 常被忽略 | - |

### 6.2 人天成本背后的"真相"

```
客户看到的：1天2000元 → 1个月6万 → "好高啊！不可能吧！"

实际构成：
 薪酬+公积金 + 年工作日200天 + 利用率80% + 项目奖金 + 房租水电
 → 人天成本远超表面数字

心智策略：给客户"算算账"
 → 客户理解成本结构后，反而产生"价值感、安全感、信任感"
 → 不敢算账的供应商，客户反而觉得"水分大"
```

### 6.3 降本三板斧

1. **差旅降本**：客户承担差旅/提供食宿/双人同住
2. **利用率提效**：减少空转，提升有效工作日占比
3. **隐性成本显性化**：把续费成本、服务器成本计入项目核算

## 七、风险防范

### 7.1 税务风险

| 风险类型 | 具体表现 | 防范措施 |
|---------|---------|---------|
| 延期开票 | 客户要求晚开票 | 按纳税义务发生时间开票，不配合客户逃税 |
| 虚开发票 | 无真实交易开票 | 红线，绝对不能碰 |

### 7.2 商业风险

| 风险类型 | 防范措施 |
|---------|---------|
| 商业贿赂 | 禁止向客户关键人输送利益，走正规商务流程 |
| 交付延期赔偿 | 合同明确交付里程碑和验收标准，延期免责条款 |

### 7.3 法律风险

- 保密义务：客户数据不出售、不泄露
- 知识产权：使用正版软件，不盗用竞品代码

## 八、SaaS vs 买断模式对比

| 维度 | SaaS（订阅模式） | 软件（买断模式） |
|------|----------------|-----------------|
| 本质 | 提供服务，一定期间内的使用权 | 销售商品，永久使用权（非所有权） |
| 交易标的 | 永远是最新版本的系统平台服务 | 特定版本，大版本升级需补差价 |
| 收费模式 | 用户数 × 周期 | 软件费 + 运维费(软件费×X%) |
| 使用权 | 付费期间有使用权 | 永久使用权 |
| 停止付费 | 无权继续使用 | 有权继续使用（厂商无权禁止） |
| 税率 | 6% | 13%（超3%即征即退，需软著备案） |

## 十、情景演练：商务谈判算账

> **案例**：客户处于商务谈判阶段
> - 你报价：产品100账号2年20万 + 实施100人天15万 = 35万
> - 客户期待：产品15万 + 实施13万 = 28万
> - 你的人天成本：1200元/天，异地需出差
>
> **算账思路**：
> 1. 产品从20万→15万：利润压缩5万，看底线
> 2. 实施100人天×1200成本=12万，报价13万→利润仅1万
> 3. 异地出差：差旅成本增加，实际可能亏损
> 4. 结论：接受28万则实施几乎零利润，需谈判重新配比（如减少实施规模、提高产品占比、让客户承担差旅）
