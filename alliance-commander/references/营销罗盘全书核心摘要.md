# 营销罗盘全书核心摘要

> 基于夏凯《营销罗盘：企业级销售体系构建与增长之道》全书提取

## 四大系统

| 系统 | 核心命题 | 对应专家 |
|------|---------|---------|
| **产业系统** | 我们在什么样的产业赛道上？市场空间多大？增长动力是什么？ | 行业洞察专家 |
| **营销系统** | 我们如何获客？如何转化？如何让销售过程可控？ | 营销规划专家、漏斗构建专家 |
| **运营系统** | 我们如何组织团队？如何管理过程？如何激发绩效？ | 组织设计专家、销售管理专家 |
| **交付系统** | 我们如何赢得客户信任？如何设计解决方案？如何持续经营？ | 信任拜访专家、解决方案专家、客户经营专家 |

## 六脉神剑（年度营销规划）

1. **市场洞察剑** — 看清市场格局，识别增长机会
2. **客户分层剑** — GMS模型：Global/Middle/Small三层分级
3. **策略制定剑** — 标品/顾问/方案/战略四种销售模式匹配
4. **组织匹配剑** — 销售组织架构与销售模式对齐
5. **过程管控剑** — 销售漏斗SOP + 赢单率预测
6. **绩效激励剑** — 销售绩效指标设计 + 激励机制

## 信任五环®（拜访方法论）

```
认知与期望 → 约定承诺 → 证实价值 → 评估意愿 → 销售回顾
    ①            ②           ③          ④          ⑤
```

核心原则：每一步都要获得客户的**行动承诺**，没有承诺的拜访就是闲聊。

## 策略九问（赢单方法论）

**识局（第1-2问）**：客户究竟要什么？我的位置在哪里？
**拆局（第3-4问）**：如何识别关键人？客户如何评价我？究竟谁说了算？
**知彼（第5-6问）**：他们到底想什么？（四个决策角色的核心诉求）
**布局（第7-9问）**：如何应对关键人？如何有效用资源？面对竞争怎么办？

四大决策角色：
- **EB**（最终决策者）— 关注ROI和战略价值
- **UB**（应用选型者）— 关注使用体验和效率
- **TB**（技术选型者）— 关注技术可行性和集成
- **Coach**（内部教练）— 关键信息源和引导者

## 5%改良模型

不要追求颠覆性改变，而是**每月5%的持续改良**：
- 本月赢单率 20% → 下月 21%
- 每月微调，12个月后赢单率提升 79%
- 聚焦可复制的行为改良，而非空洞的战略口号

## 客户金三角（经营方法论）

三个维度构建客户经营体系：
1. **覆盖度** — 有多少目标客户在管理范围内？
2. **渗透度** — 每个客户的合作深度（产品线、部门、决策层）
3. **忠诚度** — 客户续约率、增购率、推荐率
