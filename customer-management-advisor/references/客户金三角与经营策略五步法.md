## 一、客户金三角框架

### 金三角三要素

```
          客户价值
          (Value)
              │
    ┌───────────┼───────────┐
客户信任                         客户依赖
(Trust)                        (Dependency)
```

| 要素 | 定义 | 构建方法 |
|------|------|---------|
| **客户信任** | 客户愿意把业务交给你 | 兑现承诺+专业表现+长期主义 |
| **客户价值** | 客户从你这里获得的价值 | 业务价值+效率提升+成本降低 |
| **客户依赖** | 客户难以离开你 | 深度集成+数据沉淀+切换成本 |

### 金三角健康度诊断

```
信任 ⭐⭐⭐⭐⭐   ← 客户愿意推荐你？
价值 ⭐⭐⭐⭐☆   ← 客户能说清你的价值？
依赖 ⭐⭐⭐☆☆   ← 客户离开你的成本高？
```

> 诊断结论：价值认可高，但依赖度不足 → 需要深化产品集成，提升切换成本

## 二、经营策略五步法

### 五步法全流程

```
步骤1：客户分级  →  步骤2：制定经营计划  →  步骤3：执行经营动作
                                                         │
步骤5：评估改进  ←  步骤4：监控经营指标  ←───────────────────┘
```

### 步骤1：客户分级

| 级别 | 名称 | 定义 | 经营重点 |
|------|------|------|---------|
| **A级** | 战略客户 | ARR贡献前20% | 高层互访+联合创新 |
| **B级** | 重点客户 | ARR贡献20%-60% | 定期回顾+增值服务 |
| **C级** | 普通客户 | ARR贡献60%-90% | 标准化服务+自助平台 |
| **D级** | 观察客户 | ARR贡献后10% | 评估续约可能性 |

### 步骤2：制定经营计划

**一户一案模板**：

```markdown
## [客户名称] 经营计划（20XX年度）

### 一、客户现状
- ARR：XX万 | 健康度：⭐⭐⭐⭐☆
- 使用情况：[核心功能使用率]
- 关键联系人：[EB/UB/TB/Coach名单]

### 二、年度目标
- ARR目标：XX万（增量XX万）
- 健康度目标：⭐⭐⭐⭐⭐
- 增购方向：[具体产品/模块]

### 三、关键动作（按月）
| 月份 | 关键动作 | 负责人 | 预期产出 |
|------|---------|--------|---------|

### 四、风险与预案
- 风险1：[描述] → 预案：[措施]
```

### 步骤3：执行经营动作

**八大经营要素**：

| 要素 | 说明 | 执行频率 |
|------|------|---------|
| **高层互访** | 双方高管定期交流 | 季度 |
| **经营回顾** | 使用价值回顾+下一步计划 | 季度 |
| **产品培训** | 新功能培训+最佳实践分享 | 半年 |
| **行业交流** | 邀请参加行业活动/用户大会 | 年度 |
| **增值服务** | 免费健康检查/优化建议 | 季度 |
| **案例打造** | 联合打造标杆案例 | 年度 |
| **推荐获取** | 请求客户推荐/介绍新客户 | 续约时 |
| **生态连接** | 介绍客户认识生态伙伴 | 适时 |

### 步骤4：监控经营指标

**客户健康度评分卡**：

| 维度 | 权重 | 指标 | 数据来源 |
|------|------|------|---------|
| 使用深度 | 30% | 核心功能使用率 | CRM系统 |
| 使用频率 | 20% | 月活跃用户数/总用户数 | 产品后台 |
| 业务价值 | 25% | 客户自评价值评分 | 季度调研 |
| 服务满意度 | 15% | CSAT评分 | 服务记录 |
| 续约意向 | 10% | 客户明确表示续约意向 | 销售拜访记录 |

```
健康度 > 80分 → 绿色（安全）
健康度 60-80分 → 黄色（关注）
健康度 < 60分 → 红色（风险）
```

## 三、销售阶段转化模型

### 客户全生命周期

```
潜客 → 新客 → 活跃客户 → 忠诚客户 → 倡导客户
 │         │        │           │           │
 获取      启用     增购        续约        推荐
```

### 各阶段关键转化率

| 转化环节 | 基准转化率 | 优秀水平 |
|---------|-----------|---------|
| 潜客→新客 | 10% | 20%+ |
| 新客→活跃 | 60% | 80%+ |
| 活跃→忠诚 | 70% | 85%+ |
| 忠诚→倡导 | 20% | 40%+ |

## 五、客户经营工作坊五步法

### 工作坊流程

1. **客户盘点**（2小时）：列出所有客户，按ARR排序，标注健康度
2. **问题诊断**（1.5小时）：每个红色客户深度分析问题根因
3. **策略制定**（2小时）：每个A/B级客户制定"一户一案"
4. **资源匹配**（1小时）：为每个策略匹配资源（技术/市场/管理层）
5. **承诺确认**（0.5小时）：每个人确认自己的行动计划和DDL

## 六、夏凯客户经营核心洞察

1. **存量优于增量**：获取新客户的成本是保留老客户的5-25倍
2. **客户经营不是客服**：是主动的价值创造，不是被动的问题响应
3. **金三角缺一不可**：只有信任没有价值→关系户；只有价值没有依赖→随时被替代
4. **一户一案是基本功**：每个A级客户都必须有书面经营计划
5. **健康度是领先指标**：健康度下降6个月后，续约会出问题
