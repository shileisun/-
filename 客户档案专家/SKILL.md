# 客户档案专家

> 量子蜂群自带的CRM客户档案数据库，可查询客户信息、分层、区域、行业分布，支持销售跟进管理

## 触发方式

当用户需要查询CRM客户数据、客户信息、分层、区域分布、行业分布时触发此Skill。

## 功能能力

### 1. 按区域查询
支持查询以下城市：
- 杭州、宁波、温州、绍兴、嘉兴、湖州、金华、台州、衢州、丽水

示例：
- "查询杭州客户"
- "宁波有哪些客户"
- "温州客户列表"

### 2. 按优先级查询
- A类：上市公司+千人规模（1-40）
- B类：大型民企+制造业（41-75）
- C类：成长型制造企业（76-100）

示例：
- "A类客户有哪些"
- "优先级B的客户"
- "C类客户"

### 3. 按行业查询
支持行业（一级分类）：
- 制造业、服务业、贸易、建筑等

示例：
- "制造业客户"
- "服务业客户"

### 4. 按分层查询
- A类：优先级20分（高优先级）
- B类：优先级15分
- C类：优先级10分以下

示例：
- "A类客户"
- "查看B类客户"

### 5. 组合查询
支持多条件组合：
- "杭州A类客户"
- "宁波制造业客户"
- "湖州A类制造业"

### 6. 关键词搜索
支持企业名称模糊匹配：
- 搜索"青山" → 青山控股
- 搜索"正泰" → 正泰集团
- 搜索"杭叉"
