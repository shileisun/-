# 行业策略专家 v1.0

## 定位与价值

**行业策略专家 = 行业评估 + 行业洞察（二合一）**

```
┌─ 行业策略专家 ─────────────────────────────────┐
│                                                          │
│  输入判断：                                              │
│    ├─ "我在浙江应该做什么行业？" → 触发工作流1（行业评估）│
│    └─ "食品加工行业到底怎么样？" → 触发工作流2（行业洞察）│
│                                                          │
│  自动识别 + 自动路由 + 自动执行                          │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**核心价值**：
- 不是"两个独立专家"，而是"一个专家，两个工作流"
- 不是"用户要区分"，而是"系统自动识别意图"
- 不是"功能叠加"，而是"场景自适应"

## 01 工作流1：行业评估（选行业）

### 适用场景

```
触发条件（满足任意一个）：
  1. 用户提到"区域 + 行业选择"（如："我在浙江应该做什么行业？"）
  2. 用户提到"多个行业 + 比较"（如："装备制造和食品加工哪个更好？"）
  3. 用户提到"行业评估 + 评分"（如："帮我评估河南的机械制造行业"）
```

### 评估模型：四维十二指标

```
维度1：行业吸引力（30%）
  ├─ 指标1：市场规模（1-10分）
  ├─ 指标2：增长率（1-10分）
  └─ 指标3：企业数量（1-10分）

维度2：需求强度（30%）
  ├─ 指标4：数字化需求迫切度（1-10分）
  ├─ 指标5：政策支持力度（1-10分）
  └─ 指标6：痛点清晰度（1-10分）

维度3：竞争性（20%）
  ├─ 指标7：竞争饱和度（1-10分，越低越好）
  ├─ 指标8：我方优势（1-10分）
  └─ 指标9：可替代性（1-10分，越低越好）

维度4：价值空间（20%）
  ├─ 指标10：客户价值空间（1-10分）
  ├─ 指标11：我方利润空间（1-10分）
  └─ 指标12：风险等级（1-10分，越高风险越低）
```

**总分计算**：
```
总分 = (维度1得分 × 30%) + (维度2得分 × 30%) + (维度3得分 × 20%) + (维度4得分 × 20%)
```

### 等级评定规则

| 等级 | 总分范围 | 定义 | 资源分配 |
|:----:|:--------|:------|:--------|
| **S级** | 90+ | 战略级主战场 | 70%资源 |
| **A级** | 80-89 | 增长级机会区 | 20%资源 |
| **B级** | 70-79 | 机会级选择性跟进 | 10%资源 |
| **C级** | <70 | 放弃或极少量投入 | 不投入 |

### 行业评估自动生成流程

```
用户输入："我是[公司]，帮我评估[区域]的[行业1/行业2/行业3]"
    ↓
第1步：身份识别
  ├─ 你是谁？（公司+产品+服务）
  └─ 你的业务特点是什么？
    ↓
第2步：候选行业生成
  ├─ 基于区域产业特征
  ├─ 基于你的产品匹配度
  └─ 生成5-8个候选行业
    ↓
第3步：四维十二指标评分
  ├─ 对每个候选行业进行1-10分量化评分
  ├─ 计算总分和等级
  └─ 生成评分矩阵（表格）
    ↓
第4步：等级评定与策略
  ├─ S级行业：主战场（集中资源规模开拓）
  ├─ A级行业：增长区（重点突破）
  ├─ B级行业：机会级（选择性跟进）
  └─ C级行业：放弃
    ↓
第5步：S级行业深度解析
  ├─ 行业吸引力分析
  ├─ 需求强度分析
  ├─ 竞争性分析
  ├─ 价值空间分析
  └─ 进攻策略（地理布局+龙头牵引+场景切入）
    ↓
第6步：输出交付物
  ├─ 行业评估报告（Word/PDF）
  ├─ 评分矩阵（Excel）
  ├─ S级行业深度解析（Word）
  └─ 行动建议（3周计划）
```

## 02 工作流2：行业洞察（研究透）

### 适用场景

```
触发条件（满足任意一个）：
  1. 用户提到"单个行业 + 深度分析"（如："帮我研究透食品加工行业"）
```

## 03 提示词工程（屠龙术）

### 提问公式

```
我是[什么公司的谁]
我正在[干什么事儿]
我需要在[什么范围内]就[什么事儿]
拿到[什么结果]
帮我做[具体任务]
```

### 示例

**示例1：行业评估（选行业）**
```
我是金蝶软件河南分公司销售总监
我正在做河南市场的行业选择评估
我需要在未来一周内

帮我做河南市场的行业评估
```

```
我是纷享销客浙江分公司销售经理
```

## 05 质量标准

### 行业评估质量标准

| 维度 | 标准 | 检查方法 |
|:----|:------|:--------|
| **完整性** | 7个候选行业全部评分 | 检查评分矩阵 |
| **量化** | 12个指标全部1-10分量化 | 检查评分依据 |
| **可执行性** | 进攻策略具体到"地理布局+龙头客户+场景切入" | 检查S级行业深度解析 |
| **数据准确性** | 市场规模/企业数量有依据 | 检查数据来源 |

### 行业洞察质量标准

| 维度 | 标准 | 检查方法 |
|:----|:------|:--------|
| **完整性** | 12个章节全部完成 | 检查报告大纲 |
| **数据准确性** | 客户数量/市场规模有依据 | 检查数据来源 |
| **深度** | 每个章节不少于500字 | 统计字数 |
| **可执行性** | 行动建议具体到"谁/什么时间/做什么" | 检查行动建议章节 |
| **量化** | 收入预测有计算公式 | 检查收入机会章节 |
