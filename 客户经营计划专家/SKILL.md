# 客户经营计划专家 v1.0

## 定位与价值

**客户经营计划 ≠ 销售计划**

| 维度 | 销售计划 | 客户经营计划 |
|:----|:--------|:--------|
| **时间跨度** | 季度/年度 | 三年（长期主义） |
| **核心目标** | 赢单/回款 | 客户成功/持续增长 |
| **关系定位** | 供应商 | 战略合作伙伴 |
| **方法论** | 销售流程 | 战略客户经营体系 |
| **输出** | 销售行动清单 | 三年经营计划 + 关系发展地图 |

**核心价值**：
- 不是"我能卖什么"，而是"客户需要什么成功"
- 不是"短期交易"，而是"长期伙伴"
- 不是"销售视角"，而是"客户视角"

## 02 客户经营计划自动生成流程

### 工作流设计

```
用户输入："我是[公司]，帮我做[客户名称]的三年经营计划"
    ↓
第1步：客户洞察检索
  ├─ 检索已完成360度洞察
  ├─ 提取关键KPI与决策链
  └─ 确认客户行业与业务特征
    ↓
第2步：经营目标设定
  ├─ 第一年：建立信任（营收目标/关系目标）
  ├─ 第二年：深化合作（营收目标/价值目标）
  ├─ 第三年：战略伙伴（营收目标/生态目标）
  └─ 逐年营收/回款目标测算
    ↓
第3步：关系发展地图生成
  ├─ 当前关系阶段诊断
  ├─ 目标关系阶段设定（三年路径）
  ├─ 每阶段关键行动设计
  ├─ 每阶段时间节点规划
  └─ 每阶段成功标准定义
    ↓
第4步：经营体系嵌入
  ├─ 战略客户经营体系流程嵌入
  ├─ 客户分层分类管理机制设计
  ├─ 定期经营回顾机制建立（季度/半年）
  └─ 满意度管理闭环设计
    ↓
第5步：行动方案分解
  ├─ 第一年行动清单（Q1-Q4分解）
  ├─ 第二年行动清单（Q1-Q4分解）
  ├─ 第三年行动清单（Q1-Q4分解）
  ├─ 关键里程碑设定（CEO会晤/联合创新/标杆打造）
  └─ 资源需求清单（销售/技术/管理层/生态）
    ↓
第6步：风险预警与应对
  ├─ 关系风险识别（关键人变动/竞争对手介入）
  ├─ 业务风险识别（客户业绩下滑/行业变化）
  ├─ 服务风险识别（实施延期/满意度下降）
  └─ 应对策略制定
    ↓
第7步：量化指标设定
  ├─ 关系指标（拜访频次/关键人覆盖率/满意度评分）
  ├─ 业务指标（营收/回款/商机数量/赢单率）
  ├─ 价值指标（客户成功案例/标杆打造/口碑推荐）
  └─ 年度复盘机制设计
    ↓
输出：三年战略经营计划（Word/PDF，约8000字）
```

## 03 客户经营计划输出模板（逐字稿验证版）

### 案例：牧原股份三年战略经营计划（金蝶软件视角）

#### 客户洞察摘要

**客户名称**：牧原食品股份有限公司（牧原股份 002714）  
**行业**：生猪养殖（自繁自养一体化）  
**关键KPI**：PSY（26→29）/料肉比（2.8→2.5）/养殖成本（13元/kg→11.5元/kg）  
**决策链**：
- EB：董事长/总裁（关注：出栏量增长率）
- T-CB：养殖总监（关注：PSY/料肉比/养殖成本）
- T-AB：供应链总监（关注：物流成本/库存周转率）
- UB：财务总监（关注：报表周期/资金周转率）
- EK：信息总监（关注：系统集成度/IT运维成本）

**潜在商机**：
1. 智能化养殖系统（高优先级）
2. 供应链管理平台（高优先级）
3. 财务管理系统
4. 人力资源系统
5. 数据分析决策系统

---

#### 经营目标（三年）

| 年份 | 关系目标 | 营收目标 | 回款目标 | 关键里程碑 |
|:----|:--------|:---------|:---------|:--------|
| **第一年（2026）** | 建立信任（熟悉→信任） | 500万 | 400万 | 签约智能化养殖系统项目 |
| **第二年（2027）** | 深化合作（信任→同盟） | 800万 | 700万 | 扩展供应链管理+财务管理系统 |
| **第三年（2028）** | 战略伙伴（同盟→共生） | 1,200万 | 1,100万 | 打造行业标杆+联合创新 |

**三年总营收**：2,500万  
**三年总回款**：2,200万

  ├─ 关键行动：

#### 经营体系嵌入（方法论）

**战略客户经营体系流程**（嵌入经营计划）：

```
1. 客户洞察（已完成360度洞察）
   ↓
2. 经营目标设定（三年目标）
   ↓
3. 关系发展地图（路径规划）
   ↓
4. 定期经营回顾（季度/半年）
   ↓
5. 满意度管理（闭环）
   ↓
6. 价值呈现（量化ROI）
   ↓
7. 增购扩销（第二/三产品线）
   ↓
8. 口碑推荐（标杆打造+联合演讲）
```

**客户分层分类管理机制**：
- A级客户（牧原股份）：三年经营计划，季度经营回顾
- B级客户（双汇/三全/思念）：年度经营计划，半年经营回顾
- C级客户（其他）：商机管理，项目制跟进

**定期经营回顾机制**：
- 季度经营回顾（Q1/Q2/Q3/Q4）：与信息化总监+业务总监
- 半年经营回顾（H1/H2）：与总裁/董事长
- 年度经营回顾：与董事会

**满意度管理闭环**：
1. 满意度调研（每半年一次）
2. 问题识别（满意度<90分，启动改进）
3. 改进计划（15天内提交改进方案）
4. 改进执行（30天内完成改进）
5. 改进验证（满意度复测）

#### 资源需求（内部）

| 资源类型 | 第一年 | 第二年 | 第三年 |
|:--------|:--------|:--------|:--------|

#### 风险预警与应对

| 风险类型 | 风险描述 | 应对策略 |
|:--------|:--------|:--------|
| **关系风险** | 信息化总监离职/变动 | 提前建立多维关系（养殖总监/供应链总监/财务总监） |
| **关系风险** | 竞争对手介入（SAP/Oracle降价） | 强调"行业化深耕能力"+ 提前锁定三年战略合作协议 |
| **业务风险** | 牧原股份业绩下滑（猪周期） | 强调"降本增效"价值（养殖成本降低10%=年节约120亿） |
| **业务风险** | 行业政策变化（环保政策收紧） | 方案中包含"合规管理模块"，帮助客户应对政策变化 |
| **服务风险** | 项目实施延期 | 提前锁定实施资源，设定项目里程碑（每两周汇报进度） |
| **服务风险** | 客户满意度下降（<90分） | 启动满意度改进流程（15天内提交改进方案） |
