## 一、教材核心：四柱八字的结构

### 1.1 四柱
- **年柱**：祖上、父母宫、0–15 岁
- **月柱**：父母、兄弟宫、15–30 岁（最关键）
- **日柱**：自己、配偶宫
  - **日干** = 命主自己
  - **日支** = 配偶宫
- **时柱**：子女宫、晚年

## 二、教材核心：排盘步骤

### 2.1 年柱
- 用 60 甲子表查出生年的干支
- 注：年柱以**立春**为界（不是农历初一开始）

### 2.2 月柱
- 用 60 甲子表 + 出生月份
- **月支** = 节气月（不按农历）
- **月干** = 由年干推算：甲己之年丙作首、乙庚之岁戊为头、丙辛必定寻庚起、丁壬壬位顺行流、戊癸之年何方发、甲乙之上好追求

| 年干 | 正月干 |
|---|---|
| 甲己 | 丙 |
| 乙庚 | 戊 |
| 丙辛 | 庚 |
| 丁壬 | 壬 |
| 戊癸 | 甲 |

### 2.3 日柱
- 万年历查
- 60 甲子循环

### 2.4 时柱
- **时支** = 出生时辰（23-1 子、1-3 丑……）
- **时干** = 由日干推算：甲己还加甲、乙庚丙作初、丙辛从戊起、丁壬庚子居、戊癸何方发、壬子是真途

| 日干 | 子时干 |
|---|---|
| 甲己 | 甲 |
| 乙庚 | 丙 |
| 丙辛 | 戊 |
| 丁壬 | 庚 |
| 戊癸 | 壬 |

### 2.5 排大运
- 阳年（年干为甲丙戊庚壬）男命、阴年（年干为乙丁己辛癸）女命 → **顺排**
- 阴年男命、阳年女命 → **逆排**
- **起运岁数**：从出生日到下一个节气（或上一个节气）÷ 3 = 几岁起运

### 2.6 真太阳时换算
- 出生地经度 → 与 120°E 比较
- 每度差 4 分钟
- **真太阳时 = 平太阳时 + (120-经度) × 4 分钟**

## 三、教材核心：四柱宫位

### 3.1 六亲宫位
- **年柱**：祖父母、父母宫的一部分
- **月柱**：父母、兄弟
- **日支**：配偶
- **时柱**：子女

### 3.2 年龄段
- **年柱** = 0–15 岁
- **月柱** = 15–30 岁
- **日柱** = 30–45 岁
- **时柱** = 45 岁之后

### 3.3 命宫、身宫
- **命宫**：与出生月、时相关
- **身宫**：主身体强弱

## 五、教材核心：地支关系

### 5.1 六合（6 对）
- 子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合

### 5.2 六冲（6 对）
- 子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

### 5.3 三合（4 组）
- 申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金

### 5.4 三会（4 组）
- 亥子丑会水、寅卯辰会木、巳午未会火、申酉戌会金

### 5.5 三刑（4 组）
- 寅巳申刑、丑戌未刑、子卯刑、辰午酉亥自刑

### 5.6 六害（6 对）
- 子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害

### 5.7 关系影响
- **合**：牵绊、合好
- **冲**：变动、冲散

## 六、教材核心：格局

### 6.1 月令用神
- 月令 = 月支 = 用神首选
- **本气** = 主用神
- **中气** = 次用神
- **余气** = 再次

### 6.2 透干取格
- 月令本气透干 → **正格**（但要避比劫、阳刃空亡等）
- 中气透干 → 杂气格

### 6.3 八格
1. **正官格**：月令藏正官，透干
2. **七杀格**：月令藏七杀，透干
3. **正印格**：月令藏正印，透干
4. **偏印格**：月令藏偏印，透干
5. **正财格**：月令藏正财，透干
6. **偏财格**：月令藏偏财，透干
7. **食神格**：月令藏食神，透干
8. **伤官格**：月令藏伤官，透干

### 6.4 化格
- 日干与月干或时干**五合** + 化神**成方得局** → 化格
- 甲己合化土、乙庚合金、丙辛合水、丁壬合木、戊癸合火

### 6.5 从格
- **从财格**：日主无根，财星独旺
- **从官格**：日主无根，官杀独旺
- **从儿格**：日主无根，食伤独旺

## 七、教材核心：身强身弱

### 7.1 月令得气
- 日主在月令**当令** → 身强
- 日主在月令**失令** → 身弱

### 7.2 异党生扶
- 地支生扶日主 → 身强
- 天干生扶日主 → 身强

### 7.3 综合判定
- 身强：用神 = 克、泄、耗（官杀、食伤、财）
- 身弱：用神 = 生、扶（印、比劫）

## 八、教材核心：调候

### 8.1 调候原则
- **冬月**（亥子丑）→ 喜丙火
- **夏月**（巳午未）→ 喜壬癸水
- **春月**（寅卯辰）→ 喜庚金 + 丙丁火
- **秋月**（申酉戌）→ 喜壬癸水 + 丙丁火

### 8.2 调候的优先性
- **调候第一**（穷通派观点）
- **格局第二**（真诠派观点）
- 两派冲突时，**先看调候、再看格局**

## 十、教材核心：神煞

### 10.1 常用神煞
- **天乙贵人**：甲戊庚牛羊、乙己鼠猴乡、丙丁猪鸡位、壬癸蛇兔藏、辛逢马虎藏
- **驿马**：申子辰马在寅、寅午戌马在申、亥卯未马在巳、巳酉丑马在亥
- **桃花**：申子辰桃花在酉、寅午戌桃花在卯、亥卯未桃花在子、巳酉丑桃花在午
- **华盖**：申子辰华盖在辰、寅午戌华盖在戌、亥卯未华盖在未、巳酉丑华盖在丑
- **文昌**：甲乙巳午报君知、丙戊申宫丁己鸡、庚猪辛鼠壬逢虎、癸人见卯入云梯
- **天德贵人**：正月丁、二月申、三月壬、四月辛、五月亥、六月甲、七月癸、八月寅、九月丙、十月乙、十一月巳、十二月庚
- **月德贵人**：寅午戌月在丙、亥卯未月在甲、申子辰月在壬、巳酉丑月在庚

### 10.2 神煞用法
- **吉神**（天乙、文昌、天德、月德）入命 = 助
- **凶煞**（驿马、桃花、华盖）中性，看组合
- 神煞**不是断命唯一依据**，要结合五行

## 十一、教材核心：大运与流年

### 11.1 大运
- 每 10 年一步
- **天干管前 5 年、地支管后 5 年**
- 大运看时代背景

### 11.2 流年
- 当年干支
- **流年天干** = 当年外事（出名、表现）
- **流年地支** = 当年内心（情绪、变动的根源）
- **流年冲大运** = 变
- **流年合日柱** = 婚、感情

### 11.3 流月
- 当年每个月的干支
- 流月看具体发生事

## 十七、教材的核心心法

1. **基础是排盘** —— 排错了一切都错
2. **重点是格局** —— 格局定层次
3. **关键是用神** —— 用神定吉凶
4. **难点是流年** —— 流年定应事
5. **核心是实战** —— 多断、多验、多改

## 十八、教材与盲派的差异

| 项 | 浙大教材（子平派） | 盲派 |
|---|---|---|
| 来源 | 学院派 | 民间盲师 |
| 体系 | 完整理论 | 实战经验 |
| 重点 | 规则 + 体系 | 象意 + 做功 |
| 优 | 入门系统 | 断得准 |
| 缺 | 流年不够细 | 理论不系统 |

**学习建议**：**先学浙大教材（系统）打底，再学盲派（实战）加深**。
