# 价值营销专家 v2.0

## 00 夏凯价值营销心法与核心方法论（必读）

> **重要**：本专家基于夏凯老师2026年6月17日直播逐字稿深度优化，融入"心法+方法+工具"完整体系。

### 一、心法（起心动念决定成败）

#### 1.1 舍得之心 vs 得舍之心

| 类型 | 起心动念 | 结果 |
|:----|:--------|:------|
| **得舍之心**（错误） | "我做方案是为了获得客户认可，拿下订单" | 从起跑线就输了 |
| **舍得之心**（正确） | "我先成就客户，客户才会成就我" | 才能真正做好解决方案 |

**核心**：当你特别关注"客户是否认可你"的时候，你已经输了。因为你在想**你自己**，不是客户。

#### 1.2 菩萨畏因，凡人畏果

| 维度 | 凡人（销售） | 菩萨（顾问） |
|:----|:--------|:----------|
| **关注点** | 果：客户认不认可我？能不能签单？ | 因：客户的问题在哪里？我能不能帮到他？ |
| **起心动念** | 私利驱动（我想赚钱） | 利他驱动（我想帮客户解决问题） |
| **结果** | 客户能感受到你的"得舍之心"，反而不会选你 | 客户能感受到你的"舍得之心"，自然会选你 |

**金句**：
> "解决方案是先要成就对方，才会成就自己的一种做法。当你起心动念想'客户得认可我'，你已经输了。"

---

### 二、解决方案的定义（必须理解透）

```
解决方案 = 双方对确定的问题一致认可的答案，且能够带来可衡量的价值改善
```

#### 2.1 关键词解读

| 关键词 | 含义 | 常见错误 |
|:----|:------|:--------|
| **双方** | 你和客户，不是你单方面认为 | 很多人做方案是"我的套餐"，不是"双方的答案" |
| **确定的问题** | 问题因岗、因人而存在，需要诊断根因 | 上来就掏产品"大力丸"，不找问题 |
| **一致认可** | 就问题、就答案、就价值形成共识 | 不是你认可，是**双方**认可 |
| **可衡量的价值** | 价值可以量化，但不是你承诺结果 | 混淆"赋能客户"和"承诺结果" |

#### 2.2 问题因岗、因人而存在

```
厨师会不会关心木工怎么用凿子？
→ 不会！因为凿子跟厨师的岗位/职责无关。
→ 所以问题一定是某个岗位、某个人的问题。
```

**方法论**：
1. 先找**岗位角色**（财务总监/生产总监/销售总监...）
2. 再找**职责目标**（每个岗位有5-7个KPI）
3. 再找**现状差距**（现状 vs 目标 = 问题）
4. 再找**关键问题**（每年重点考核2-3个指标，这就是关键问题）

### 三、价值营销四步法（与客户共同完成）

```
发现 → 定义 → 设计 → 共识（不是你单方面做，是与客户一起做）
```

| 步骤 | 内容 | 输出 |
|:----|:------|:------|
| **发现** | 锚定业务目标和关键问题 | 客户痛点清单 |
| **定义** | 定义关键问题和问题产生的原因 | 问题根因分析 |
| **设计** | 设计可行的路径、问题的解决方法 | 解决方案初稿 |
| **共识** | 对问题、方法、路径、价值形成共识 | 双方认可的答案 |

**注意**：不是你上来就说"我知道你的病怎么治，然后开方"，那是你的答案，不是双方的答案。

### 四、目标→障碍→能力→价值（8字诀）

> 这是价值营销的**灵魂**，吃透这8个字，前途不可限量。

```
目标（Goal） → 障碍（Obstacle） → 能力（Capability） → 价值（Value）
```

#### 4.1 逻辑链条

| 步骤 | 问题 | 目的 |
|:----|:------|:------|
| **目标** | 客户的关键角色有什么目标？（KPI） | 锚定靶心 |
| **障碍** | 阻碍目标实现的关键问题是什么？（根因） | 诊断问题 |
| **能力** | 如何通过产品/服务赋予客户能力？ | 设计方案 |
| **价值** | 客户应用能力后创造什么量化价值？ | 计算ROI |

#### 4.2 实例：生产部部长（产品合格率从98%→99%）

| 步骤 | 内容 |
|:----|:------|
| **目标** | 产品合格率从98%提升到99% |
| **障碍** | 设备故障预警不及时、工艺参数波动、人员操作不规范 |
| **能力** | 我们的MES系统赋予客户"实时监控+预警+工艺参数优化"的能力 |
| **价值** | 产品合格率提升1%，年减少质量损失约500万 |

**关键**：你卖的是"棍子"，客户拿棍子去"打狼"。你**不对结果负责**，你只对"赋能客户"负责。

### 五、赋能客户（灵魂概念）

> **你不对客户的结果和价值负责，你只对"赋能客户"负责。**

#### 5.1 棍子与狼的比喻

```
客户要上山打狼。
你卖给客户一条棍子。
问题：客户买了棍子，能不能一定打死狼？
回答：不一定！

正确心态：
- 你卖的是棍子（产品/解决方案）
- 客户拿棍子去打狼（创造价值的主体是客户）
- 你赋能客户具备"打狼的能力"（提供棍子+培训+赋能）
- 你不对"打死狼"这个结果负责（那是客户的事）
```

#### 5.2 赋能值 vs 承诺值

| 类型 | 含义 | 示例 |
|:----|:------|:------|
| **赋能值** | 你的产品/服务赋予客户的能力 | "我们的MES系统赋予您实时监控能力" |
| **承诺值** | 你承诺客户能达到的结果 | "我们的MES系统保证您的产品合格率提升1%" ❌（不能这么承诺） |

**金句**：
> "客户要上山打狼，你给他卖了一条棍子。请问你给他卖的这条棍子，客户拿着棍子上山能不能打死狼，不一定吧？但是你是卖棍子的。"

### 六、价值矩阵（5类价值+可控度分析）

#### 6.1 5类价值

| 价值类型 | 含义 | 示例 |
|:----|:------|:------|
| **潜在价值** | 客户可能获得的价值 | "提升竞争力、提升利润"（不可控） |
| **锚定价值** | 与客户共识的价值目标 | "库存周转天数从45天降至30天" |
| **成本节约价值** | 降低成本带来的价值 | "年节约库存资金占用约1,000万" |
| **效率提升价值** | 提升效率带来的价值 | "财务结账周期从50天缩短至25天" |
| **收入增长价值** | 增加收入带来的价值 | "销售预测准确率提升，减少断货损失约500万/年" |

#### 6.2 价值可控度分析

| 可控度 | 含义 | 示例 | 能否写入方案？ |
|:----|:------|:------|:----------|
| **高可控** | 价值可以直接量化、与客户确认 | 库存周转天数缩短、财务结账周期缩短 | ✅ 可以 |
| **低可控** | 价值间接、受多因素影响 | 竞争力提升、利润提升、客户满意度提升 | ⚠️ 谨慎（要说明假设条件） |

**原则**：
- 高可控价值 → 大胆写，有计算公式
- 低可控价值 → 谨慎写，说明"假设条件"

### 七、沟通心法：共情→诊断→根因→建议

#### 7.1 错误示范（卖大力丸）

```
客户："我们最近老丢单，想提升赢单率。"
销售（条件反射）："我有赢单九问！我帮你提升赢单率！"（❌ 像卖大力丸）
```

#### 7.2 正确示范（共情→诊断）

```
客户："我们最近老丢单，想提升赢单率。"
销售（共情）："王总，我理解，像咱们这种复杂解决方案销售，客户决策链复杂，
              提升赢单率确实是很重要的事。确实挺难的。"（✅ 同理心）

销售（诊断-现状）："王总，我想了解一下，咱们目前团队多少人？主要面向哪些客户？
              大概订单规模是多少？"（✅ 了解现状）

销售（诊断-根因）："王总，在您看来，目前影响我们赢单率的要素有哪些？
              我们现在是怎么做的？"（✅ 找根因）

销售（建议）："王总，我有个建议，我们可以建立一个规则，每个大项目必须
              有coach、有需求清晰度分析、有决策链视图..."（✅ 形成共识）
```

#### 7.3 四季沟通法（信任五环）

| 季节 | 沟通重点 | 目的 |
|:----|:--------|:------|
| **春（共情）** | 理解客户难处，建立同理心 | 让客户觉得"你懂我" |
| **夏（诊断）** | 诊断现状、诊断根因 | 让客户觉得"你专业" |
| **秋（建议）** | 提供建设性方案和建议 | 让客户觉得"你有方法" |
| **冬（共识）** | 对问题、方案、价值形成共识 | 让客户觉得"这就是我要的" |

### 八、专业信任建立（客户愿不愿意跟你说实话？）

#### 8.1 核心问题

```
客户不跟你说实话，是谁的问题？
→ 是你的问题！（事有不成，反求诸己）
```

#### 8.2 值得托付的人设

| 维度 | 做法 | 目的 |
|:----|:------|:------|
| **专业度** | 展现对业务的理解、对问题的洞察 | 让客户觉得"你懂行" |
| **可靠感** | 兑现承诺、做事严谨 | 让客户觉得"你可以信赖" |
| **共情度** | 理解客户难处、不急于推销 | 让客户觉得"你值得托付" |

**金句**：
> "客户特别想跟懂他的人、值得托付的人敞开心扉。你是谁？你是卖的，客户就不会跟你探讨他的问题。"

| 检查项 | 标准 | 自查 |

## 定位与价值

**价值营销 ≠ 产品推销**

| 维度 | 产品推销 | 价值营销 |
|:----|:--------|:--------|
| **出发点** | 我有什么产品 | 客户有什么需求 |
| **沟通重点** | 产品功能/特性 | 客户KPI/业务价值 |
| **价值呈现** | 定性描述 | 量化数据 |
| **客户关系** | 交易型 | 合作伙伴型 |
| **价格谈判** | 降价/打折 | 价值论证/ROI |

**核心价值**：
- 不是"我能卖什么"，而是"客户需要什么价值"
- 不是"产品功能介绍"，而是"KPI提升量化"
- 不是"价格谈判"，而是"ROI论证"

## 01 价值营销五步法（核心方法论）

### 五步法流程

```
第一步：客户有什么需求？
   ↓   （基于客户洞察）
   ├─ 显性需求：客户主动表达的痛点
   └─ 隐性需求：客户未意识到但存在的痛点
   
第二步：哪些岗位涉及？他们的KPI是什么？
   ↓   （角色-指标映射）
   ├─ 决策角色识别（EB/T-CB/T-AB/UB/EK）
   ├─ 每个角色的KPI提取（行业自适应）
   └─ KPI baseline与target梳理
   
第三步：我们有什么产品/能力？
   ↓   （能力匹配）
   ├─ 产品功能清单
   ├─ 行业化解决方案
   └─ 差异化优势
   
第四步：给每个岗位创造什么价值？
   ↓   （价值主张）
   ├─ 每个KPI对应的价值点
   ├─ 价值点量化（提升X%/降低Y%/缩短Z%）
   └─ 差异化价值主张卡
   
第五步：价值如何量化？
   ↓   （数据驱动）
   ├─ ROI计算公式
   ├─ 3年收益测算
   ├─ 回本周期计算
   └─ 输出：价值营销方案 + ROI分析
```

## 02 价值营销自动生成流程

### 工作流设计

```
用户输入："我是[公司]，帮我做[客户名称]的价值营销方案"
    ↓
第1步：客户认知
  ├─ 检索客户洞察（已完成的360度洞察）
  ├─ 提取客户需求清单（显性+隐性）
  └─ 确认客户行业与业务特征
    ↓
第2步：角色-KPI映射
  ├─ 识别决策角色（EB/T-CB/T-AB/UB/EK）
  ├─ 提取每个角色的KPI（行业自适应）
  ├─ 梳理KPI baseline与target
  └─ 输出：角色-KPI矩阵
    ↓
第3步：能力匹配
  ├─ 检索产品/能力清单
  ├─ 匹配客户需求与产品功能
  ├─ 识别差异化优势
  └─ 输出：能力匹配矩阵
    ↓
第4步：价值主张生成
  ├─ 每个KPI对应的价值点
  ├─ 价值点量化（提升X%/降低Y%）
  ├─ 生成差异化价值主张卡
  └─ 输出：价值主张卡（每角色1张）
    ↓
第5步：ROI量化
  ├─ 选择ROI计算公式（基于场景）
  ├─ 输入参数（投资额/年收益/成本节约）
  ├─ 计算3年ROI与回本周期
  └─ 输出：ROI分析报告
    ↓
第6步：方案整合
  ├─ 整合所有价值主张卡
  ├─ 整合ROI分析
  ├─ 生成价值营销方案（Word/PDF）
  └─ 输出：价值营销方案 + ROI分析 + 价值主张卡
   
耗时：约60秒
```

## 03 价值营销输出模板（逐字稿验证版）

### 案例：牧原股份价值营销方案

#### 客户信息

**客户名称**：牧原食品股份有限公司（牧原股份 002714）  
**行业**：生猪养殖（自繁自养一体化）  
**规模**：年出栏量8,000万头，营收1,200亿  
**关键KPI**：PSY/料肉比/养殖成本/出栏率/毛利率

---

#### 第一步：客户需求清单

**显性需求**（客户主动表达）：
1. 智能化养殖系统升级（降低养殖成本）
2. 供应链管理平台（提升物流效率）
3. 财务管理系统升级（缩短报表周期）

**隐性需求**（客户未意识到但存在）：
1. 数据孤岛打通（现有用友财务+SAP供应链+华为智慧化不互通）
2. 业财一体化（业务数据与财务数据实时同步）
3. 大数据分析决策系统（从"经验决策"到"数据决策"）

---

#### 第二步：角色-KPI矩阵

| 角色 | 岗位 | 关键KPI | Baseline | Target | 提升空间 |
|:----|:----|:--------|:-------|:------:|:--------|
| **EB** | 董事长/总裁 | 出栏量增长率 | 15%/年 | 20%/年 | 5% |
| **T-CB** | 养殖总监 | PSY（断奶仔猪数） | 26 | 29 | +12% |
| | | 料肉比 | 2.8 | 2.5 | -10% |

##### ROI计算

```
总投资：4,800万元（3年）
3年总收益：约4,161,000万元

ROI = (3年总收益 - 总投资) / 总投资 × 100%
    = (4,161,000 - 4,800) / 4,800 × 100%
    ≈ 86,587%

回本周期 = 总投资 / 年度收益
          = 4,800 / 1,387,000
          ≈ 0.0035年
          ≈ 1.3天
```

**注意**：以上收益测算基于牧原股份超大规模（年出栏8,000万头），实际ROI会更高。

对于中型企业（年出栏100万头），ROI测算：
```
总投资：500万元（3年）
3年总收益：约2,000万元

ROI = (2,000 - 500) / 500 × 100%
    = 300%

回本周期 = 500 / 666.7
          ≈ 0.75年
          ≈ 9个月
```

## 04 ROI量化公式库

### 通用公式

#### 1. 人效提升收益

```
年收益 = 年营收 × 人效提升比例 × 销售占比

示例：
年营收：100亿
人效提升比例：10%
销售占比：60%

年收益 = 100亿 × 10% × 60% = 6亿
```

#### 2. 费用节约收益

```
年收益 = 年营收 × 费用率 × 下降比例

示例：
年营收：100亿
费用率：8%
下降比例：15%

年收益 = 100亿 × 8% × 15% = 1.2亿
```

#### 3. 客户流失减少收益

```
年收益 = 活跃客户数 × 流失率 × 平均客单价 × 挽回率 × 留存提升

示例：
活跃客户数：10,000
流失率：10%
平均客单价：10万
挽回率：50%
留存提升：20%

年收益 = 10,000 × 10% × 10万 × 50% × 20% = 1,000万
```

#### 4. DSO缩短现金流收益

```
年收益 = (原DSO - 新DSO) / 365 × 年营收 × 资金成本率

示例：
原DSO：90天
新DSO：60天
年营收：100亿
资金成本率：5%

年收益 = (90 - 60) / 365 × 100亿 × 5% ≈ 411万
```

#### 5. 服务效率提升收益

```
年收益 = (原处理时间 - 新处理时间) / 原处理时间 × 服务人员成本

示例：
原处理时间：2小时
新处理时间：0.5小时
服务人员成本：1,000万/年

年收益 = (2 - 0.5) / 2 × 1,000万 = 750万
```

#### 6. 库存周转率提升收益

```
年收益 = (新周转率 - 原周转率) / 原周转率 × 库存金额

示例：
原周转率：6次/年
新周转率：7.2次/年
库存金额：50亿

年收益 = (7.2 - 6) / 6 × 50亿 = 10亿
```
