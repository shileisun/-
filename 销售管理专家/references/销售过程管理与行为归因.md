# 销售过程管理与行为归因（夏凯体系延伸）

> 来源：夏凯《营销罗盘》体系 + 纷享销客销售管理实践
> 适用：销售管理专家 / 过程监控 / 行为归因 / 偏差诊断

---

## 一、销售管理在营销罗盘中的位置

### 营销罗盘四大系统关系

```
市场系统 ──→ 销售系统 ──→ 交付系统
   ↑                      ↓
   └──────── 组织系统 ←────┘
         (销售管理在这里)
```

**核心观点**：销售管理不是"盯结果"，是"管过程"。结果是滞后的，过程是可以实时干预的。

---

## 二、销售过程管理体系

### 三层过程指标

```
┌─────────────────────────────────────────┐
│      结果指标（滞后，月度看）              │
│  ARR完成率、签单金额、回款金额          │
├─────────────────────────────────────────┤
│      行为指标（实时，周度看）              │
│  拜访次数、决策人覆盖度、承诺获取数        │
├─────────────────────────────────────────┤
│      活动指标（先行，日度看）              │
│  通话时长、邮件往来、方案发送、POC推进   │
└─────────────────────────────────────────┘
```

### 指标因果链

```
活动指标 → 行为指标 → 结果指标
  (因)        (果/因)      (果)
```

**管理要点**：结果不好→回溯行为指标→再回溯活动指标，找到根因

## 三、行为归因诊断法

### 归因五问

当销售业绩不达标时，逐层追问：

```
Q1：结果指标是否达标？
    → 否，进入Q2
    
Q2：行为指标是否达标？
    → 拜访次数够吗？决策人覆盖全吗？承诺获取了吗？
    → 若有未达标行为，进入Q3
    
Q3：活动指标是否支撑行为？
    → 每天通话时长够吗？邮件往来频繁吗？
    → 若活动不足，是能力问题还是态度问题？
    
Q4：能力和态度问题，根因是什么？
    → 培训不足？激励不到位？工具不好用？
    
Q5：组织系统是否需要调整？
    → 流程有问题？资源不匹配？文化不支持？
```

### 归因矩阵

| 症状 | 归因层级 | 解决方案 |
|------|---------|---------|
| 拜访次数够，但转化率极低 | 能力问题（L2） | 销售技能培训 |
| 活动量很高，但拜访约不到 | 方法问题（L2） | 电话销售技巧训练 |
| 转化率不错，但商机量太少 | 市场问题（L1） | 增加线索投入 |
| 所有人业绩都差 | 系统问题（L3） | 组织系统诊断 |
| 个别人业绩差 | 态度/能力问题（L2） |  coach或淘汰 |

## 四、销售管理专家的三类核心工作

### 工作1：过程指标监控

**周度销售会议模板**：

```markdown
## 销售周会（[日期]）

### 一、数据回顾（15分钟）
- 本周活动量：[通话] [拜访] [方案发送]
- 本周新增商机：金额XX万，数量X个
- 本周赢单：XX万，XX个
- 本周输单分析：原因 [列举]

### 二、商机逐单过堂（30分钟）
按赢率排序，重点讨论：
- 赢率>70%：确认签单时间，推进合同
- 赢率30-70%：诊断卡点，制定下一步
- 赢率<30%：评估是否继续投入

### 三、行为归因（10分钟）
- 活动量异常的销售：[姓名] → 原因？
- 转化率异常的商机：[客户] → 根因？

### 四、下周计划（5分钟）
- 重点跟进商机：
- 重点支持需求：
```

### 工作2：异常预警与干预

**预警规则（示例）**：

| 异常类型 | 触发条件 | 自动动作 |
|---------|---------|---------|
| 商机停滞 | 同一阶段>30天 | 提醒销售+经理介入 |
| 拜访缺失 | 连续5工作日无拜访记录 | 提醒销售 |
| 决策人缺失 | 商机无EB联系人 | 提醒补充决策链 |
| 承诺缺失 | 拜访后无下一步承诺 | 触发教练辅导 |
| 输单预警 | 竞品出现+赢率下降 | 升级策略赢单专家 |

## 五、销售阶段转化模型（监控版）

### 各阶段健康度基准

| 阶段 | 平均停留时间 | 阶段转化率基准 | 异常信号 |
|------|------------|---------------|---------|
| 线索确认 | <7天 | 40% | >14天未推进 |
| 初步沟通 | <14天 | 50% | 3次沟通无进展 |
| 需求确认 | <30天 | 60% | >45天无需求文档 |
| 方案验证 | <45天 | 50% | POC超时无结果 |
| 商务谈判 | <30天 | 70% | 合同条款反复修改 |
| 合同签订 | <15天 | 90% | 合同签署拖延 |

### 转化漏斗健康度评分

```
漏斗健康度 =
  (各阶段停留时间达标率 × 0.3) +
  (各阶段转化率达标率 × 0.4) +
  (行为活跃度达标率 × 0.3)
```

## 六、销售管理专家的KNOW HOW

### 何时调用

- 销售业绩异常，需要诊断根因
- 销售过程不透明，需要建立监控体系
- 销售行为数据丰富，需要归因分析
- 需要设计销售KPI体系和激励机制

### 核心交付物

1. **销售过程监控看板**（实时，含异常预警）
2. **行为归因诊断报告**（个人/团队维度）
3. **销售阶段转化分析**（找瓶颈环节）
4. **销售改进计划**（个人发展计划）
5. **销售管理SOP**（过程管理标准化）

### 质量自检清单

```
□ 过程指标是否覆盖了活动→行为→结果全链条？
□ 预警规则是否及时（而非滞后）？
□ 归因分析是否找到了可行动的根因？
□ 改进计划是否有具体的检查点？
□ 销售管理SOP是否与激励机制对齐？
```

## 七、夏凯核心洞察（销售管理维度）

1. **管过程，不等结果**：结果是滞后的，过程中就能干预
2. **行为数据是领先指标**：今天的行为→下个月的结果
3. **归因要穿透三层**：活动→行为→系统，找到真正的根因
4. **销售管理不是管控**：是coach和销售赋能
5. **数据是起点，不是终点**：看到数据→诊断→行动，才是完整闭环
