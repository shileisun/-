## 定位

在量子蜂群Pipeline中，销售漏斗构建专家（v2.0）位于「营销规划」之后、「销售策略」之前——承接市场策略和客户分类结果，构建标准化的销售流程体系和漏斗管理框架。

```
专家规划 MTL（谋略司·流程建设层）：
  营销规划专家 → 销售漏斗构建专家（v2.0） → 销售策略专家
      ↓                                  ↓
  行业洞察专家 → 客户洞察专家 → 商机挖掘专家
                      ↓
  营销组织设计专家 → 生态专家

专家赢单 LTC（执行层）：
  销售策略专家 → 客户拜访专家 → 解决方案专家 → 客户经营专家
      ↑_______________________________________________↑
                   销售管理指导专家（全程监控）
```

## 目的

为销售团队提供标准化的流程建设工具，包括：漏斗阶段设计、转化率分析、可视化看板、赢单预测、CRM数据对接、SOP流程制定。

---

## 一、销售漏斗五大作用

1. **购买逻辑**：体现客户思维认知过程、群体决策形成过程
2. **购买流程**：按照"意向→方案→评估→决策"等步骤进行采购
3. **指导路径**：按照统一步骤和方法进行销售，提供标准化流程
4. **成交概率**：通过不同阶段的进程、要素、质量等因素定义赢单率
5. **管理工具**：根据漏斗形状、流速、斜率等指标分析预测业绩

## 二、销售漏斗阶段模型

### 按单产金额分层

| 单产金额 | 销售周期 | 常规赢单率 | 销售模式 |
|---------|---------|-----------|---------|
| 3万元以下 | 2-4周 | 3%-5% | 标品在线销售 |
| 3-10万元 | 4-8周 | 5%-10% | 电销或单兵配套 |
| 10-30万元 | 1.5-3个月 | 10%-15% | 单兵或售前支持 |
| 30-80万元 | 3-6个月 | 15%-20% | 销售售前协同 |
| 80-200万元 | 3-9个月 | 20%-25% | 团队协同销售 |
| 200-800万元 | 6-12个月 | 25%-30% | 团队协同销售 |
| 800-2000万元 | 9-18个月 | 30%-35% | 铁三角销售 |
| 2000万元以上 | 12个月以上 | 35%-40% | 重装旅协同销售 |

**关键洞察**：单产金额越高，销售周期越长，赢单率越高，需要的销售团队协同程度越高

## 三、销售漏斗分析框架

### 分析维度

1. **漏斗形状**：各阶段数量分布是否合理
2. **漏斗流速**：各阶段转化时间是否正常
3. **漏斗斜率**：相邻阶段转化率是否健康
4. **关键阻塞点**：识别转化率最低的阶段

### 提升策略

1. 提升市场到线索转化率
2. 提升线索到商机转化率
3. 提升商机到成交转化率
4. 缩短销售周期
5. 提升每单单产

## 四、赢单因素分析

### 外部因素

- **目标市场选择**：选择公司有优势的目标市场
- **客户采购过程**：了解客户思维模式、关注重点和决策逻辑
- **市场竞争**：应对激烈竞争和非常规策略
- **客户内部意见**：处理不同部门、不同主管领导之间的分歧

### 内部因素（核心突破点）

- **团队无兵法**：缺乏统一的销售方法论
- **个体缺能力**：销售人员未经过正式训练，缺乏关键能力
- **组织无标准**：缺乏有效的商机评估和跟进机制
- **资源不精准**：资源调动不及时、不精准

## 七、AI赢单预测模块

### 赢单概率计算公式

```
基础赢单概率 = (各维度得分 x 权重) x 10

决策链分x0.25 + 需求匹配分x0.20 + 竞争态势分x0.20 + 价值认知分x0.15 + 时间紧迫分x0.10 + 预算充足分x0.10
```

### 各维度评分标准

| 维度 | 权重 | 评估要点 | 评分标准（0-10分） |
|------|------|---------|-------------------|
| **决策链完整度** | 25% | EB/TB/UB各层级是否覆盖 | 完整=10；覆盖2层=6；覆盖1层=3；无覆盖=0 |
| **需求匹配度** | 20% | 产品功能与需求匹配程度 | 完全=10；基本=7；部分=4；不匹配=0 |
| **竞争态势** | 20% | 我方竞争优势 | 领先=10；持平=6；落后=3 |
| **价值认知** | 15% | 客户对方案价值的理解 | 高度认同=10；基本认同=6；不足=3 |
| **时间紧迫性** | 10% | 采购时间窗口 | 紧迫(<=30天)=10；一般=6；不紧迫=3 |
| **预算充足性** | 10% | 预算覆盖方案 | 充足=10；基本=6；不足=3 |

### 阶段调整系数

```
赢单概率 = 基础赢单概率 x 阶段调整系数

需求调研: 0.9 | 方案制定: 1.0 | 商务谈判: 1.1 | 即将签约: 1.3
```

### AI预测话术
- "预测 [商机名称] 的赢单概率"
- "评估 [客户] 的商机健康度"
- "给出 [商机] 的提升建议"

## 九、销售SOP流程图

### 标准LTC流程图

```
线索获取 → 线索验证(有效→商机确认, 无效→归档)
         → 需求调研 → 需求匹配(匹配→方案制定, 不匹配→方案调整)
         → 商务报价 → 商务谈判(达成→合同签署, 未达成→再次谈判)
         → 交付实施 → 验收回款 → 客户经营 → 增购续费/流失预警
```

### 各阶段关键动作

| 阶段 | 关键动作 | 完成标准 | 时长要求 |
|------|---------|---------|---------|
| 线索获取 | 市场活动/名单获取 | 有效线索定义 | - |
| 线索验证 | 电话沟通/信息确认 | 确认需求意向 | 3天内 |
| 商机确认 | 初访/需求确认 | 填写商机卡 | 1周内 |
| 需求调研 | 深度访谈/现场调研 | 完成调研报告 | 2周内 |
| 方案制定 | 方案撰写/内部评审 | 方案审核通过 | 1周内 |
| 商务报价 | 报价单发出/议价 | 商务条款确认 | 1-2周 |
| 合同签署 | 合同审批/签章 | 合同盖章生效 | 1周内 |
| 交付实施 | 项目启动/执行 | 按时交付 | 按项目周期 |
| 验收回款 | 验收确认/回款 | 款项到账 | 按合同约定 |

## 🔥 v13.0 新增：张睿CRM落地策略——全生命周期漏斗分层

> **来源**：张睿《CRM解决方案落地策略V5.2》
> **核心观点**：漏斗不是一条线，而是四阶段分层管理——每个阶段有独立的漏斗逻辑

### 一、客户全生命周期四阶段漏斗

```
潜在客户阶段（MTL漏斗）         意向客户阶段（LTC漏斗）
┌──────────────────┐      ┌──────────────────┐
│ 目标客户定义      │      │ ☆评级体系筛选     │
│   ↓ ABM营销获取   │      │   ↓ 项目立项跟进   │
│ 客户画像标签      │      │ 商机阶段管理       │
│   ↓ 动态数据管道  │      │   ↓ 产销协同       │
│ 列名客户入库(CRM) │  →   │ 赢单/输单          │
└──────────────────┘      └──────────────────┘
         ↓                        ↓
  成交客户阶段（MCR漏斗）    流失客户阶段（ITR漏斗）
┌──────────────────┐      ┌──────────────────┐
│ 忠诚度指标体系    │      │ 流失预警监测      │
│   ↓ 分层经营策略  │      │   ↓ 挽回五步法    │
│ 健康度模型监控    │  ←   │ 流失客户管理      │
│   ↓ 增购/续费     │      │   ↓ 七项关键举措  │
│ 客户成功          │      │ 恢复/放弃         │
└──────────────────┘      └──────────────────┘
```

### 二、意向客户☆评级漏斗

> 漏斗不只是"数量递减"，更是"质量递增"——用☆评级做漏斗质量过滤器

| 漏斗层级 | ☆评级 | 客户数占比 | 转化目标 | 管理动作 |
|:---:|:---:|:---:|:---:|------|
| **漏入** | 所有意向客户 | 100% | — | 统一建档、七问采集 |
| **过滤层1** | ≥2☆ | ~60% | 进入培育池 | 补充缺失维度信息 |
| **过滤层2** | ≥5☆ | ~35% | 进入正式跟进 | 关键决策人建联、需求确认 |
| **核心层** | ≥8☆ | ~15% | 进入商机推进 | 铁三角资源倾斜、加速推进 |

### 三、ABM工具路径（全链路漏斗）

```
列名客户入库(CRM) → 营销活动获取动态数据(CRM)
→ 个性化和前瞻性建议(市场舆情匹配)
→ 实现ABM营销策略工具(纷享销客CRM)
→ 销售流程自动化获取动态数据(CRM应用)
```

**各阶段关键指标**：
| 阶段 | 核心指标 | 基准值 |
|:---:|---------|:---:|
| 列名客户→活跃线索 | 营销触达率 | ≥60% |
| 活跃线索→意向客户 | MQL转化率 | ≥20% |
| 意向客户→商机 | SQL转化率 | ≥15% |
| 商机→签约 | 赢单率 | ≥25% |

### 三、业务结果四阶段理论

**漏斗构建专家在四阶段中的位置**：
- **第一阶段（结果清晰化）**：漏斗诊断 → 让销售清楚"问题在哪里"
- **第二阶段（结果量化）**：赢单指数计算 → 让销售看到"改进能带来什么"
- **第三阶段（结果达成路径）**：漏斗优化方案 → 让销售看到"怎么改进"
- **第四阶段（结果闭环）**：过程管理SOP → 让销售清楚"每天做什么"
