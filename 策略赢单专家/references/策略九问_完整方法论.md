## 二、识局篇（看清形势）

### 第一问：客户究竟要什么？

| 分析维度 | 关键问题 |
|---------|---------|
| 需求范畴 | 需求范畴和目标清晰吗？ |
| 认知归属 | 这是客户的认知还是我的认知？ |
| 选择空间 | 客户还有选择的其他方案吗？ |
| 环境变化 | 识别客户内外部环境的变化 |

**客户反应四类**：

| 类型 | 特征 | 策略 |
|------|------|------|
| **如虎添翼型（Growth）** | 现状不差，但愿意追逐梦想 | 展示增值价值，描绘愿景 |
| **亡羊补牢型（Trouble）** | 出了问题，急于补救 | 直击痛点，快速响应 |
| **两厢情愿型（Need）** | 有明确需求，理性选择 | 匹配需求，差异化竞争 |
| **锦上添花型（Opportunity）** | 有机会但紧迫度低 | 培育需求，长期经营 |

**SSO（单一销售目标）检查清单**：
- 跟我有关系吗？
- 清晰具体吗？
- 有预算吗？
- 有期限吗？
- 有竞争吗？

> "没有差异，就没有采购。" "销售目标是'定盘星'，一切行动围绕目标展开。"

---

### 第二问：我的位置在哪里？

**形势三维结构**：销售进程 × 客户紧迫程度 × 竞争形势

| 分析维度 | 关键问题 |
|---------|---------|
| 阶段判断 | 项目到了什么阶段？ |
| 踩刹车 | 什么时候该踩刹车？ |
| 漏斗误导 | 销售漏斗是否会误导判断？ |
| 紧迫程度 | 客户的紧迫程度怎么样？ |
| 竞争态势 | 我们在竞争中处于什么位置？ |
| 项目温度 | 如何感知项目温度？ |

---

### 第三问：如何识别关键人？

**"帅马车炮"角色模型（象棋类比）**：

| 象棋角色 | 对应销售角色 | 特征 | 关注点 |
|---------|------------|------|-------|
| **帅** | EB（最终决策者） | 一锤定音 | 投资回报、组织影响 |
| **马** | UB（应用选型者） | 灵活走位 | 工作方式、绩效影响 |
| **车** | TB（技术选型者） | 直来直去 | 技术标准、合规安全 |
| **炮** | Coach（教练/内线） | 隔山打牛 | 帮助销售制定策略 |

> 还要识别局里的**"潜伏者"**——隐藏在表面之下的影响者，以及**EK把关者**（合规/流程/审批）。

## 三、拆局篇（厘清关系）

### 第四问：客户如何评价我？

| 分析维度 | 关键问题 |
|---------|---------|
| 变化影响 | 变化导致形势变化？ |
| 态度决定 | 角色态度决定成败？ |
| 支持程度 | 如何衡量支持程度？ |
| 态度标识 | 如何标识客户态度？ |
| 优劣标识 | 如何标识优势与风险？ |
| 结构分析 | 分析态度不能忽略结构？ |

**态度标识体系**：

| 标识 | 含义 | 等级 |
|------|------|:---:|
| **G+4** | 强烈支持，主动推动 | 最高 |
| **G+3** | 支持，愿意推荐 | 高 |
| **G+2** | 倾向支持 | 中高 |
| **G+1** | 略有好感 | 中 |
| **T(0)** | 中立/未表态 | 中 |
| **EK-1** | 有顾虑/质疑 | 中低 |
| **EK-2** | 反对/抵制 | 低 |
| **EK-3** | 强烈反对/设障碍 | 最低 |

---

### 第五问：究竟是谁说了算？

| 分析维度 | 关键问题 |
|---------|---------|
| 小角色影响 | 小角色有多大影响？ |
| 参与规律 | 多人参与的规律？ |
| 影响力来源 | 影响力由什么决定？ |
| 潜在博弈 | 洞察潜在的博弈力量？ |

**影响力公式**：影响力 = 职位权力 × 专业权威 × 信息控制 × 关系网络

---

### 第六问：客户到底想什么？

| 价值维度 | 分析要点 |
|---------|---------|
| 组织价值 | 效率提升、成本降低、业务增长 |
| 个人价值 | 职业发展、工作便利、个人声誉 |
| 隐性理由 | 决策的隐性动机是什么？ |
| 决策动力 | 如何寻找决策动力？ |
| 双赢可能 | 这次销售能双赢吗？ |

> **关键**：不仅要看表面价值，更要挖掘决策的隐性理由和决策动力。

## 四、布局篇（疏通要点）

### 第七问：如何应对关键人？

| 角色 | 应对策略 | 关键动作 |
|------|---------|---------|
| **EB（帅）** | 投资回报 + 战略价值 | 行业对标案例、ROI量化、高层互访 |
| **UB（马）** | 工作绩效 + 易用性 | 试用体验、用户案例、成功故事 |
| **TB（车）** | 技术标准 + 合规 | 技术白皮书、POC验证、方案评审 |
| **Coach（炮）** | 识别→信任→使用 | 获取内部情报、制定精准策略 |
| **EK把关者** | 合规流程 + 风险 | 合规文档、流程说明、法务对接 |
| **消极客户** | 激发需求意识 | 展示同业案例、制造紧迫感 |
| **自满客户** | 揭示隐患风险 | 现状差距分析、风险预警 |

**Coach发展三步法**：
1. **识别**：谁愿意主动提供信息？谁对你的方案感兴趣？谁在内部有影响力？
2. **建立信任**：专业能力展示、诚实可靠、保护Coach身份
3. **使用Coach**：获取内部信息、引荐关键人、验证策略假设

---

### 第八问：如何有效用资源？

| 资源类型 | 具体内容 | 运用原则 |
|---------|---------|---------|
| **公司资源** | 专家、案例、资金、高管 | 按需申请，用在刀刃上 |
| **客户资源** | 内部教练、需求信息 | 善用Coach获取内部支持 |
| **生态资源** | 合作伙伴、渠道、行业联盟 | 借力打力，扩大影响 |

**资源申请三原则**：
1. 明确资源用途和预期效果
2. 说明不用资源的风险和损失
3. 提供资源使用的具体方案

---

### 第九问：面对竞争怎么办？

| 竞争态势 | 策略 | 关键动作 |
|---------|------|---------|
| **我方领先** | 巩固策略 | 强化优势、加速推进 |
| **双方持平** | 差异化策略 | 找到独特价值、不可替代 |
| **我方落后** | 反超/侧翼策略 | 找差距补短板、绕开正面 |
| **对手根深蒂固** | 渗透策略 | 从边缘入手、逐步突破 |
| **陷入价格战** | 价值策略 | 避免大幅杀价、回归客户价值 |

> **核心原则**：以客户为中心而非对手为中心。警惕"陶醉"——自满是危险的开始。

## 五、核心定律总结

- **策略销售与销售漏斗**的关系：漏斗管"量"，策略管"质"
- **什么是专业化销售**：不是靠关系、靠价格，而是靠策略、靠专业
- **成功销售的必备素质**：系统思维 + 结构化分析 + 灵活应变 + 持续学习
