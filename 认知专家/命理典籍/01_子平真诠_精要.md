# 《子平真诠》精要 · 沈孝瞻

> 源出：清代沈孝瞻。子平术"格局派"的集大成之作，被徐乐吾誉为"命理学最完善之著作"。

## 一、核心思想
子平真诠的**第一原则**：**用神配气候，月令为纲**。
- 一命之主 = **月令用神 + 配气候**（寒暖燥湿）
- 格局成败 = **月令所藏用神能否透干 + 是否有情有力**

## 二、八格（正格）
月令所藏透干者为格：
- **正官格** / 七杀格
- **正印格** / 偏印（枭神）格
- **正财格** / 偏财格
- **食神格** / 伤官格
- （比劫不成格，但月令比劫可取）

## 三、格局成败——"成格"与"破格"

### 成格的条件
1. **月令用神透干**（如寅月甲木透干为比肩格，但若甲透又见庚来克，须有制）
2. **相神有力**（相神 = 辅格之神）
3. **用神不被冲克合化坏**
4. **日主能任**（身能担财官）

### 破格的条件
1. **用神被冲克合化**（如正官格遇伤官见官）
2. **相神受伤**（如财格用食生财，印克食则破）
3. **格局不清**（多格混杂，主次不分）

## 四、用神与相神

**用神**：一命之"主"（最关键的那个字）
**相神**：辅格之神（决定格局高低）
**忌神**：克用神者
**仇神**：克相神者
**喜神**：生用神、助相神者

> "用神一成，格局已定；相神一伤，格局破败。"

## 五、官杀、印财、食伤的"清纯"问题
- 官杀混杂 → 须合杀留官或合官留杀
- 印枭混杂 → 须去枭留印
- 财印交战 → 看月令谁旺
- **总原则：清纯 > 杂**

## 六、论日主强弱
子平真诠**不完全按旺衰派**断强弱，而是：
- **月令用神** = 第一要紧
- **格局成败** = 第一原则
- 日主强弱 = 第二考虑

> 故真诠有"身弱用印、可以成格"之论，**非一味扶抑**。

## 七、核心断语
1. "**有情有力**"——格局成
2. "**无情无力**"——格局破
3. "**月令为纲**"——断命首看月令
4. "**相神为辅**"——相神受伤，祸不旋踵
5. "**清纯为贵**"——杂则浊

## 八、变格论
真诠除八格外，论**专旺格**（一行得气）、**从弱格**（无根弃命）、**化格**（甲己合化土等）。

---

## 实战要点（麻辣笔记）
- 看真诠：**先定月令用神 → 再看透干 → 再看相神 → 再看日主能否担**
- **真诠最大特色**：讲"格局成败"的**具体条件**，可操作性强
- 弱点：论六亲、看流年偏简略（这是盲派补的地方）

> **麻辣心得**：真诠是子平术"内功心法"，重"格"；盲派是"实战拳法"，重"象"。两者配着看最准。
