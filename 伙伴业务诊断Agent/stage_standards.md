## 活跃客户判定标准（来自工具③）

| 类型 | 判断条件 |
|:---:|------|
| 🔴 **必选项** | 一年内有IT产品采买记录（CRM、ERP、OA等） |
| 🟠 **可选项1** | 近期有重组、并购、IPO或组织变革 |
| 🔵 **可选项2** | 近期有请咨询公司做过战略咨询、销售咨询等 |
| 🟢 **可选项3** | 高成长性企业（专精特新小巨人、隐形冠军等） |

**判定**：必选项 + 至少一个可选项 = **活跃客户**

## 各阶段发展路径

### 初创启动期（0-3年）
- **重心**：活下来。开源是第一要务
- **业务动作**：FABM开源（ABM精准名单+定向触达）、微信添加、聚焦CRM标准版、小单快跑积累案例
- **团队配置**：创始人1人 + 兼职/实习1人
- **核心风险**：老板转身赤字、现金流断裂

### 规模发展期（3-5年）
- **重心**：建立可复制打法
- **业务动作**：电话+生态双管齐下、微信运营、选1-2行业试点、建立初步交付流程
- **团队配置**：销售负责人1人 + 销售1-3人 + 实施1人
- **核心风险**：打法不可复制、续费率低

### 专业经营期（5年以上）
- **重心**：品牌+效率
- **业务动作**：行业深耕、存量经营、生态合作、团队赋能
- **团队配置**：销售总监1人 + 销售经理2-3人 + 销售3-5人 + 实施3-5人 + 技术2-3人 + 市场1人
- **核心风险**：增长放缓、行业聚焦不坚定

### 战略共赢期
- **重心**：生态整合
- **业务动作**：战略绑定、生态领军、全国布局、产品创新
- **团队配置**：总经理1人 + 销售10-20人 + 实施8-15人 + 研发5-10人 + 市场2-3人 + 职能3-5人

## 两大经营原则

1. **聚焦原则**：选择最有效的业务打法，同时放弃非核心业务。假设 → 验证 → 建模 → 规模 → 生态。
2. **长板原则**：基于自身长板选择业务模型，而非补短板。

## 建议赋能重心（来自工具③）

根据诊断结果选择不超过3项：
- 战略聚焦辅导 / 算账能力培训 / 组织体系搭建 / 执行机制设计
- 价值销售培训 / 干部力培育 / 经营会升级 / 复盘机制建立
