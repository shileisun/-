## 一、2467月度经营体系

```
2号 → 4号 → 6号 → 每周
出报表   出报告   共识会   任务令
  ↓       ↓       ↓       ↓
数据可视  分析可视  决策可视  进度可视
表      告      会      令
```

| 节点 | 内容 | 关键要求 |
|------|------|---------|
| **2号** | 出报表 → 数据可视 | 快：当天出；准：主数据治理；狠：指标有效 |
| **4号** | 出报告 → 分析可视 | 用华为五段论结构，看全看深找内因 |
| **6号** | 共识会 → 决策可视 | 目标与能力平衡，形成决策和任务令 |
| **每周** | 任务令 → 进度可视 | 三驾马车管理，说到做到 |

**核心原则**：2号报表未出，不开会；4号报告未出，不讨论分析。

## 二、四可视闭环

### 1. 数据可视（快·准·狠）

**前提**：明确责任中心——开好经分会的第一步是明确谁对什么负责。

**核心公式**：收入 - 成本 - 费用 = 利润

**收入四大来源**：

| 来源 | 含义 | 提升方向 |
|------|------|---------|
| 流量获取 | 进来了多少人 | 拓渠道、扩开源 |
| 转化率 | 进来的人多少买了 | 提能力、强打单 |
| 客单价 | 买了多少 | 价值销售、方案化 |
| 复购率 | 买了又买多少 | 客户经营、续费增购 |

**核心驱动**：战略大单品 + 组织扩展

### 2. 分析可视（看全·看深·找内因）

**分析四步法**：
① 列出理想值 vs 实际值 → ② 计算GAP值（执行差距+目标差距）→ ③ 分析原因，聚焦内因 → ④ 共识Roadmap

**两个核心视角**：

| 视角 | 对比 | 目的 |
|------|------|------|
| 🟦 执行差距 | 实际 vs 目标，差距在哪？ | 提升能力（面向过去） |
| 🟩 目标差距 | 机会点在哪？还能打多远？ | 挖潜能（面向未来） |

**分析三原则**：
1. 把问题上墙，聚焦问题本身
2. 尽量找内因——外部环境难改、内部能力可建
3. 看全看深——避免片面、断层、短期、单点视角

**抓机会比找问题更重要。**

### 3. 决策可视（主观符合客观）

**OODA循环**（适用于不确定环境——当前经济环境千变万化，必须从PDCA转向OODA）：

```
O → O → D → A
观察   判断   决策   行动
```

| 对比 | OODA | PDCA |
|------|------|------|
| 全称 | 观察→判断→决策→行动 | 计划→执行→检查→改进 |
| 数据位置 | 数据在前，先观察再判断 | 数据在后，先计划再检查 |
| 适用场景 | 不确定环境（战场、变化市场） | 稳态环境（制造业、确定性流程） |

**目标与能力平衡**：
- 太激进：目标远超能力
- 太保守：有能力不敢打，错过窗口

**能力建设先后序**：人财物销 → 数字化/流程 → 战略

**利润格局判断**：

| 格局 | 收入 | 利润 | 判断 |
|------|:---:|:---:|------|
| 健康增长 | ↑ | ↑ | 目标与能力匹配 |
| 提质增效 | ↓ | ↑ | 主动收缩，聚焦利润 |
| 🔴 增收不增利 | ↑ | ↓ | 规模虚胖，利润失血 |
| 🔴 双降 | ↓ | ↓ | 失去增长动力 |

### 4. 进度可视（说到做到）

**两个到位**：
- **说到位**：任务描述完整——谁做、做什么、什么标准、什么时候完成
- **做到位**：执行彻底——不折不扣、不拖延、不打折

## 五、人财物销 · 能力构建先后序

**放量前必须回答**：有没有优秀的团队？有没有优秀的干部？→ 没有就不要搞规模。

```
人 > 财 > 物 > 销
↓     ↓     ↓     ↓
干部   算清   产品  有利润
是第一  账才  有购  卖出去
位     能赚   买理
       钱     由
```

| 能力 | 关键点 | 风险 |
|------|--------|------|
| 人 | 干部是第一位，招高潜人才 | 没有人，只有老板分管这块业务 |
| 财 | 区分会计与财务，数据参与经营 | 不会算账→卖的越多亏的越多 |
| 物（产品） | 从"生"到"养"，IPMT做商业决策 | 产品不行→坑的越多客户越知道 |
| 销 | 产品有购买理由，有利润卖出去 | 销售很强但产品不行→卖越多坑越多 |

**华为原则**：以客户为中心，以生存为底线——服务客户要算账，不赚钱就是在坑客户。

**会计 vs 财务**：数据参与经营 = 财务，否则 = 会计。

**牛鼻子**：产品做好 + 钱分好。

## 六、营销四P + 产品决策

### 营销四P：三个出口一个入口

| P | 本质 | 关键词 |
|:--:|------|--------|
| 产品 | 购买理由 | 投钱 |
| 渠道 | 利益分配机制 | 分钱 |
| 推广 | 品牌与流量 | 烧钱 |
| **定价** | **入口**（定价定天下） | — |

### 产品决策：从"生"到"养"

| 生（盲目） | 养（有规划） |
|-----------|------------|
| 一夜激情，疯狂出产品 | 聚焦养大，需要克制 |
| 资源有限，生得越多养不起 | 需要组织来决策 |
| 穷养，自生自灭 | IPMT判断值不值得做 |
| 老板见客户就想做 | 技术委员会做预研 |

**IPMT**：商业决策组织，判断做不做——未来能赚多少、要投多少、多少时间。

## 七、战略思维三要点

### ① 品类思维
竞争的目的是消灭竞争。未来十年，在哪个细分市场成为品类之王？

```
品类之王 → 定价权 → 高毛利 → 招优秀人才
```

### ② 爬楼梯摘果子
构建核心能力做楼梯。华为：ICT能力（楼梯）→ 运营商（第1棵树）→ 企业网络（第2棵树）→ 终端/汽车（第3棵树）

> 大家都在地面上拼命跑，内卷极其厉害。但爬得高的人，不需要去抢地面的果实。

### ③ 持久战 + 速决战
战略要持久（十年成为品类之王），战斗要速决（每一场集中优势兵力）。

## 八、先画圈，再划线

**第一步：画圈（界定问题）**
- 问题是什么？
- 边界在哪？
- 谁是责任主体？

**第二步：划线（解题思路）**
- B/G/R/A 维度分解
- 形成行动路径
- 标准动作重复做

> 案例：健身——锁定大肌肉群（画圈）→ 五个标准动作 → 每个动作10-12组（标准动作重复做）。经营也一样。

## 十一、月度经营会闭环

### 会议纪律
1. 2号报表未出，不开会；4号报告未出，不讨论分析
2. 只讨论数据和决策，不讲故事、不找借口
3. 每项决策必须落到任务令
4. 会议不超过90分钟
5. 会后24小时内发出会议纪要

### 闭环检查
- 上月任务令是否全部回顾？
- 本月决策是否每项都有责任人+截止时间？
- 是否每项都已开出任务令？
- 遗留问题是否有下月目标？
- 下次会议时间是否确认？

### 任务完成率统计
按时完成/延期完成/未完成/完成率四柱统计。

### 遗留问题
连续2个月未闭环的遗留问题，必须升级为专项任务令，指定专人限期解决。

## 十二、财务经营底层逻辑（DAY1：从会算账到能挣钱）

> 📌 **主讲**：郭保彬，2026年5月，26页PPT
> 📌 **核心命题**：从"知道怎么算账"到"真正能赚到钱"，中间隔着一整套经营思维的转变
> 📌 **详细内容**：参见 `finance_to_profit.md`

### 12.1 赚钱认知 · 三大报表的经营者视角

| 报表 | 一句话 | 经营者必问 |
|------|--------|-----------|
| 利润表 | 我这一年到底赚没赚钱？ | 营业利润还是靠其他收益？ |
| 资产负债表 | 我的家底到底有多厚？ | 资产负债率多少？ |
| 现金流量表 | 钱到底去哪了？ | 经营现金流是正还是负？ |

**5大认知误区**：规模大=实力强(×) → 规模有质量的：利润 / 有利润=不缺钱(×) → 利润有质量的：现金流 / 融资越多越好(×) → 有财务风险 / 企业资产=老板身价(×) → 别忘了负债 / 企业钱=老板钱(×) → 法律主体不同

**权责发生制**：成本费用在受益期间内平均分配。折旧/计提/摊销是"被忽略的利润杀手"。

**收入 ≠ 收款**：SaaS订阅按月分摊确认，实施按投入法确认，客开按生命周期分摊，对接验收时一次性确认。收入不含税（合同收入 = 合同金额÷1.06）。

**利润 ≠ 现金收入 - 现金支出**：利润表有自己的结构（毛利→营业利润→利润总额→净利润）。

### 12.2 算大账 · 代理商长期财务预算模型

**产出三层递进**：ARR规模 → 合同收款 → 返款收入  
**投入结构**：销售(HU+提成) / 交付(HU+提成) / CSM(HU+提成) + 销售费用 + 固定费用 + 税费  
**利润拆解**：产品利润 + 交付利润 + 续费利润 - 公共费用 - 增值税 = **现金利润**

**人效金线**：人均收款(HU合计÷整体收款)，是检验组织效率的第一指标。

### 12.3 算小账 · 项目级利润

```
人天成本单价 = 人均年薪 ÷ (年工作日 × 有效利用率)
项目利润 = 返款收入 - 人工成本 - 提成 - 差旅 - 税
```

**必查清单**：提成覆盖/实施成本/差旅承担/税率/隐性投入

### 12.4 SaaS经营逻辑

**代理商盈利模型**：以利润为导向，以规模为牵引。核心公式：**新购率 + 增购率 > 流失率**。

**ARR增长陷阱**：随着ARR规模增大，即便90%续费率，流失绝对值也在增长。NGR<5%即陷入增长停滞。NGR+NDR十年模拟：NGR 10%/NDR 100% → 12倍增长；NGR 0%/NDR 80% → 几乎不增长。

**订阅 vs 实施返款**：持续续费前提下，订阅长期收益更高。每1元实施费=本金，续费每年40%收益率。

**客户生命周期**：续费率80%→5年，90%→10年，95%→20年（LTV=1÷流失率）。

### 12.5 高质量签约

| 标准 | 量化 | 含义 |
|------|:---:|------|
| 有利润 | 返款覆盖提成+分摊 | 没利润的合同是负债 |
| 能耗低 | 实施÷ARR ≤ 2 | 实施太重，长期不划算 |
| ARR占比高 | 订阅÷总额 ≥ 60% | 续费才是长期价值 |

**正确客户三原则**：以商业利益为前提 / 做愿意为价值付费的客户 / 做甜区客户

### 12.6 成本管控 · 七大隐性成本

人工刚性成本 / 续费成本(实施费20%) / 服务器运维 / 增值税(合同额4.5%) / 房租物业 / 差旅费 / 第三产品

**降本三板斧**：差旅客户承担 / 提升利用率 / 隐性成本纳入项目核算

**心智策略**：给客户算账 → 客户反而觉得"有安全感"→ 不敢算账的供应商客户觉得"水分大"

### 12.7 风险防范

**税务**：虚开发票(红线)/延期开票(按纳税义务时间)  
**商业**：商业贿赂(走正规商务流程)/交付延期(明确里程碑+免责条款)  
**法律**：保密义务/知识产权

### 12.8 赚钱机会（七大方向）

1. 企业Agent底座和数据中台
2. 多元化创新产品
3. 构建Agent市场
4. ToB产品ToC销售
5. 客户感知价值提升
6. 直播模式
7. 行业场景Agent产品创新（价值实现型 + 价值辅助型）

### 12.9 SaaS vs 买断模式

| 维度 | SaaS(订阅) | 软件(买断) |
|------|-----------|-----------|
| 本质 | 服务 | 商品 |
| 使用权 | 付费期间有 | 永久有 |
| 税率 | 6% | 13% |
