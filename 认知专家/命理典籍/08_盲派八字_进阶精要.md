### 1.2 盲派"金木水火土死期诀"

```
金怕临官木怕衰，土逢沐浴来降灾。
火到死地无有救，水流绝地回不来。
金衰木绝土怕养，水病火死不久长。
```

**口诀释义**：
- **金**：行"临官"运最怕（庚金见寅卯地）
- **木**：行"衰"运最怕（甲乙木见申酉地）
- **土**：行"沐浴"运最怕（戊己土见子午地，桃花地）
- **火**：行"死"运无救（丙丁火见亥子地）
- **水**：行"绝"运回不来（壬癸水见巳午地）

## 二、断父母亡期·盲派口诀

### 2.1 断父亲死亡时间

```
比劫重重虽克父，不见偏财命也固。
岁运若遇偏财来，父死非命无救助。
偏财临库又见冲，岁运见库父寿终。
天冲地冲见偏财，父亡之期提前来。
年干受克父先死，年食月枭父早埋。
年柱纳音被月克，岁运同克父必亡。
偏财临旺克神衰，旺极岁运父死埋。
```

**口诀释义**：
- 年干克（被月干、日干、时干克）→ 父先死
- 年柱纳音被月柱纳音克 → 父寿终
- 偏财入墓又逢冲 → 父寿终
- 大运流年同来克 → 父亡期

### 2.2 断母亲死亡时间

```
财多无印母无凶，岁运逢印母必亡。
天地转杀逢印绶，母命必然不长寿。
年支受冲母早亡，年印月财母必伤。
正印遇墓被刑冲，再临墓库母寿终。
天克地冲伤正印，无救之时母被埋。
官杀死绝财特旺，岁运逢之母亡期。
四柱印旺财星衰，旺极岁运母死埋。
```

**口诀释义**：
- 财多无印本无凶，但**岁运逢印反为忌**（母星被冲）
- 正印入墓 + 刑冲 → 母寿终
- 天克地冲 + 印被伤 → 无救则母被埋

### 2.3 断能否送终父母

```
父母送终看克冲，冲克必定能送终。
相生相合不见面，四柱五行来推验。
虽冲冲神遭逆克，最后可能不送终。
```

> 解读：命局与父母星相冲相克 = 能送终；相生相合 = 难见最后一面。

### 2.4 断父母谁先亡

```
若问父母谁先亡，月令年柱细审详。
年干为父支为母，五行生克记心上。
月令克谁谁先死，相生相比寿命长。
```

> 简明：月令五行克年干/支，谁被克谁先亡。

### 2.5 断父母诀·盲派六亲活变

> 盲派核心思维：**"六亲活变"——把六亲当太极点重起盘**

```
判定父母首重位，分清男女是应当。
男干为父支为母，女干为母支为父。
日干定出何属性，一生必是围它行。
另外参考旺与衰，下定断语必确精。
倘若详细断父母，要把父母作太极。
重新确定盘与命，简简单单便断命。
```

**核心要点**：
- 男命：年干 = 父、年支 = 母
- 女命：年干 = 母、年支 = 父
- **要断父母之事，把父母星当太极，重新起盘**：日干对父星（男看偏财、女看正印）的生克关系决定关系好坏：
  - 父生日干 → 慈爱有加
  - 日干生父 → 孝顺
  - 日干克父 → 忤逆
  - 父克日干 → 严管当情
