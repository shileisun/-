## 模型总览

| 业务模型 | 适用阶段 | 核心逻辑 |
|---|---|---|
| FABM开源 | 所有阶段 | 基于ABM方法论精准锁定目标客户，以行业/规模/画像筛选+定向触达为准 |
| 生态开源 | 所有阶段（初创期特别适用） | 锁定金蝶/用友/MES类核心管道，建立人脉圈，批量获取生态伙伴线索 |
| 销售力 | 所有阶段 | 客户接触→初次拜访→需求调研→解决方案报价→POC验证，每个阶段采用不同作战路径 |

## FABM开源（所有阶段适用）

> FABM = Fxiaoke Account-Based Marketing，纷享销客版ABM精准获客体系。
> 核心理念：不是广撒网打电话，而是用矛叉鱼——先锁定目标客户画像，再精准触达。

### 三策略矩阵

| 策略 | 适用场景 | 目标客户数 | 投入深度 |
|------|------|:---:|:---:|
| Strategic ABM (1:1) | 大客户/标杆客户 | 3-5家 | 深度定制方案+高层关系 |
| ABM Lite (1:Few) | 行业头部客户群 | 10-30家 | 行业方案+精准触达 |
| Programmatic ABM (1:Many) | 目标行业泛客户 | 50-200家 | 标准化内容+规模化运营 |

### 核心指标

| 指标 | 标准值 |
|---|---|
| 目标客户名单精准度（行业/规模/区域匹配） | ≥ 80% |
| 日定向触达量（电话+微信+邮件+LinkedIn） | 30-50次 |
| 日新增微信好友 | 10+ |
| 3个月商机转化率 | 5-10% |
| 已触达客户中产生商机比例 | ≥ 15% |

### 四步操作流程

```
① 名单锁定 → ② 画像研究 → ③ 精准触达 → ④ 培育转化
```

### 操作要点

1. **名单锁定（重中之重）** —— 基于行业、规模、区域、数字化成熟度四维筛选精准目标客户名单。不是"可能需要的企业"，而是"一定需要的企业"。使用FABM客户数据库和企查查/天眼查交叉验证
2. **画像研究** —— 对每个目标客户做1页纸画像：业务特征/核心痛点/决策链/采购历史/竞争格局。投入越深，转化越高
3. **精准触达** —— 不是冷呼叫，而是"我已经研究过你了"的专业开场。电话+微信+行业社群+线下活动四步递进。每次触达有目的、有内容、有后续
4. **内容弹药库** —— 准备行业白皮书/案例集/ROI计算器/竞品对比表等专业内容，每次触达带内容，不带推销
5. **CRM全程追踪** —— 所有触达记录、画像信息、商机阶段录入CRM，形成可追踪的ABM Pipeline

### 核心心法

> ABM不是"找客户"，是"选客户"。选对了客户，销售就成功了一半。
> 名单精准度 > 触达数量。100个精准触达 > 1000个盲目电话。

## 生态开源（所有阶段，初创期特别适用）

### 核心数据

- 40%左右的商机来源于核心管道
- 每天能够加到 **10个伙伴**，跟 **5个伙伴** 能够进行联系

### 核心管道选择

聚焦 **金蝶、用友、MES** 类的核心管道。不管在哪个阶段，特别是初创期的伙伴，要把精力放在生态开源上面。

### 操作要点

1. **核心管道选择很重要** —— 一定要聚焦核心管道，宜少不宜多，逐步深入最好
2. **选择有客户基础的伙伴** —— 选择那种有客户、有老客户的，与自己的资源能够匹配的
3. **微信获取线索三种方式**：
   - ① 加上微信，相互介绍，给到对方自己的产品
   - ② 建一个群，把他拉入到自己群里面来运营
   - ③ 每周或每月有 **2-3次** 的互动
4. **持续办线下人脉会** —— 每月至少办一次线下市场活动，采用线下人脉会加一对一的交流方式
5. **拓展下级生态** —— 飞书、钉钉等平台，在某一伙伴圈内、某一区域内深度经营，形成人脉口碑、市场口碑、伙伴口碑

## 销售力（所有阶段）

### 销售流程五步法

```
客户接触 → 初次拜访 → 需求调研 → 解决方案报价 → POC验证
```

### 递增回报法则

1. **市场占有率是第一位的** —— 基础客群规模决定口碑影响力和高阶客户转化量
2. **高阶客群由基础客群而来** —— 基础客群越大，可升级的高阶客户也越多
3. **新增客户由既有客户带来** —— 专业领域的客户选购具有转介绍效应

### 高效成交SOP（详见附录A）

- 假设→验证→建模→规模→生态（业务模型五阶段）
- 单一销售目标（SSO）管理
- 信任五环（认知与期望→约定承诺→证实价值→评估意愿→销售回顾）

## 市场活动（成长发展期及以上）

### 核心指标

| 指标 | 标准值 |
|---|---|
| 办会频次 | 月度 |
| 单场规模 | ≤ 20人 |
| 3个月转化率 | 30~40% |
| 市场效应 | 持续 |

### 操作要点

1. **聚焦签约转化** —— 规模20人以内，整体可控。签约转化是唯一衡量标准
2. **会前精准邀约** —— 对目标参会客群做销售分析梳理，重点商机在会前亲自上门邀约
3. **内容专项设计** —— 课件要经过专项设计，甚至办会前一晚根据参会名单做内容侧重设计
4. **内部团队PK** —— 邀约数、意向数、预测签约数上墙，组织小组PK，每日积分战报
5. **关键价值** —— 不是现场成单（现场概率极低），而是3个月的持续转化效应
6. **素材二次利用** —— 每次打单会做图文稿在自有平台和公域发布；视频录制剪辑，在抖音、视频号、小红书多平台矩阵发布

### 操作流程

```
会前2周：名单梳理·邀约
    ↓
会前1周：上门拜访·过堂
    ↓
会中：内容呈现·促签约
    ↓
会后：跟进转化·复盘
```

## 行业化3.0根据地建设（所有阶段）

### 核心逻辑

聚焦 **1-3个行业** 吃透，符合纷享销客客户画像三大行业，树立样板。

### 三个阶段

1. **行业选择** —— 选择有客户基础、有案例、有认知的行业
2. **样板建设** —— 打造1-2个标杆客户，形成行业化3.0根据地
3. **规模复制** —— 以样板为武器，规模化复制推广

### 适用阶段

- 初创期：选择1个行业试点
- 成长发展期：聚焦1-2个行业
- 规模发展期：聚焦1-3个行业
- 专业经营期：深耕1-3个行业
- 战略共赢期：全国布局，多行业深耕

## 存量客户经营（有客户基础的阶段）

### 核心指标

| 指标 | 初创期 | 成长发展期 | 规模发展期 | 专业经营期 | 战略共赢期 |
|---|---|---|---|---|---|
| 客户续约率 | ≥ 60% | ≥ 65% | ≥ 75% | ≥ 85% | ≥ 90% |
| 增购/转介绍占比 | ≤ 5% | 5-15% | 15-25% | 25-40% | ≥ 40% |

### 操作要点

1. **客户成功体系建设** —— 保续费、挖增购、促转介绍
2. **定期回访机制** —— 每月至少1次回访，每季度至少1次深入交流
3. **增购机会挖掘** —— 从单一产品到多产品，从标准版到旗舰版
4. **转介绍机制** —— 满意客户是最好的销售，建立转介绍激励政策
5. **客户分层经营** —— 按ARR金额和客户战略重要性分层，配置不同服务资源

### 客户成功三个核心变量

1. **产品价值持续交付** —— 客户是否持续获得价值
2. **客户关系深度** —— 是否建立了多维度的客户关系
3. **续约流程管理** —— 续约是否提前3个月启动

## 六度业务模型选择原则

### 好的业务打法三条件

1. **有方法套路，有流程** —— 告诉你第一步做什么，接下来做什么，需要投入多少资源
2. **有数据** —— 知道模型的输入输出数据，能判断是不是一个合理的增长模型
3. **有样板** —— 已有伙伴通过这套打法取得了成功，具备可复制性

### 业务模型发展五阶段

```
假设 → 验证 → 建模 → 规模 → 生态
```

每个阶段做不同的事：
- **假设**：提出业务假设，定义目标客户和的价值主张
- **验证**：小范围测试，验证假设是否成立
- **建模**：将验证成功的打法抽象为可复制的模型
- **规模**：将模型规模化复制，形成稳定产出
- **生态**：将模型开放给生态伙伴，形成生态效应

## 任务令管理（来自工具②）

> 📌 **来源**：经盈工作坊工具②任务令模板

### 任务令结构

一张任务令包含以下要素：
1. **基础信息**：任务名称、负责人、截止日期、优先级（P0/P1/P2）
2. **任务背景**：为什么要做（背景/目的/与公司目标的关联）
3. **任务内容**：具体要做什么（拆成可执行的子任务，每条一件事）
4. **完成标准**：成功长什么样（用具体数字/可验证的结果描述）
5. **资源与支持**：人力支持、预算、预计风险
6. **关键里程碑**：时间节点、里程碑内容、负责人、完成情况
7. **进度更新**：更新日期、进展说明、下一步行动、更新人
8. **任务结项**：实际完成日期、最终交付物、经验与改进

### 任务令三驾马车

| 类型 | 说明 | 面向 |
|:---:|------|:--:|
| ① 常规流程类 | 现有能力可完成的事 | 过去+现在 |
| ② 提升能力类 | 补短板、提效率 | 过去 |
| ③ 机会抓取类 | 抓新机会、新方向 | 未来 |

### 任务令使用规则

1. 每项重要工作都开任务令，一事一令
2. 发令人与负责人当面确认完成标准后才算生效
3. 每周例会前负责人主动更新进度，不等老板问
4. 完成后必须结项，写下改进心得，沉淀为公司经验
5. 没有任务令的决策不算决策

## 周会标准结构（来自工具④）

> 📌 **来源**：经盈工作坊工具④周会标准结构卡

### 核心原则

不汇报、不流水账，只解决三类问题：**进展（绿）、风险（黄）、阻塞（红）**。

### 七段式结构

| 环节 | 颜色 | 内容 |
|:--:|:--:|------|
| 一、参会签到 | — | 提前2分钟确认，迟到标注 |
| 二、本周进展 | 🟢 绿 | 有结果的事，只说结果不展开过程 |
| 三、本周风险 | 🟡 黄 | 可能影响目标的事，必须有人跟进 |
| 四、本周阻塞 | 🔴 红 | 需要上级/跨部门支援，当场确认支援人 |
| 五、下周计划 | — | 每人报1-3件最重要的事 |
| 六、周会闭环 | — | 风险跟进/阻塞支援/计划确认/下次时间 |
| 七、会议备注 | — | 主持人填写，发送全员 |

### 周会纪律
- 准时开始，迟到记录
- 每人发言≤2分钟，主持人计时
- 汇报≠讨论：只说结果和需求
- 会议≤60分钟，超时散会
- 阻塞问题当场确认支援人，24小时内跟进

## 下半年经营计划一页纸（来自工具⑤）

> 📌 **来源**：经盈工作坊工具⑤

### 结构（七大板块）

| 板块 | 内容 |
|:--:|------|
| 一、战略聚焦 | 不超过3句话：聚焦哪个行业/客群？核心价值是什么？ |
| 二、核心目标 | 新签/续费/综合收入/活跃客户/续费率/团队规模，上下半年对比+增长率 |
| 三、季度拆解 | Q3/Q4目标+重点+关键战役（1-2场） |
| 四、资源配置 | 人怎么分（销售/实施/客服/管理）+ 钱怎么分（人力/获客/培训/运营） |
| 五、TOP3风险 | 最大风险 + 发生概率（高/中/低） + 应对预案 |
| 六、TOP3行动 | 不是目标，是打法。每项行动：负责人+截止时间 |
| 七、制定审核 | 老板签字 + 渠道干部审核 + 下次review时间（建议每月一次） |

### 使用规则
1. 制定后3天内发渠道干部备案
2. 每月周会上对照检查，偏差超过20%立即调整
3. 季度末做完整复盘
4. 不是写给老板看的，是写给自己执行的

## 月度经营会2467体系（来自工具⑥ + 可视化经营分析会）

> 📌 **来源**：经盈工作坊工具⑥ + 可视化经营分析会PDF

### 2467节奏

| 节点 | 产物 | 核心要求 |
|:--:|------|---------|
| **2号** | 出报表 → 数据可视 | 快·准·狠。当天出，指标有效 |
| **4号** | 出报告 → 分析可视 | 华为五段论。看全看深找内因 |
| **6号** | 共识会 → 决策可视 | 目标与能力平衡，主观符合客观 |
| **每周** | 任务令 → 进度可视 | 说到做到 |

### 月度经营会六大板块

| 板块 | 内容 |
|:--:|------|
| 一、数据仪表盘 | 6项核心指标（收入/新签/续费/新客/续费率/毛利率）+ 环比 + 完成率 |
| 二、利润格局 | 健康增长/提质增效/增收不增利/双降，四象限判定 |
| 三、分析可视 | 华为五段论速填 + GAP分析（执行差距+目标差距） |
| 四、决策可视 | 目标与能力平衡校验 + 本月决策清单（每项→任务令） |
| 五、进度可视 | 上月任务令闭环 + 完成率统计（按时/延期/未完成/完成率%） |
| 六、会议闭环 | 回顾/决策/任务令/遗留/下次时间，五项确认 |

### 倒闭三步曲警戒线（每期必查）

```
增收不增利 → 盈亏平衡 → 资不抵债
  (收入↑利润↓)   (踩线边缘)     (危险)
```

若触及盈亏平衡线，当月必须出专项应对方案。

### 月度经营会纪律
1. 2号报表未出，不开会；4号报告未出，不讨论分析
2. 只讨论数据和决策，不讲故事、不找借口
3. 每项决策必须落到任务令
4. 会议≤90分钟
5. 会后24小时内发出会议纪要，含所有决策项和任务令编号
