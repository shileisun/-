# 线索营销专家 v1.0

## 定位与价值

**线索营销专家 = 线索开发 + 价值营销（二合一）**

```
┌─ 线索营销专家 ─────────────────────────────────┐
│                                                          │
│  输入判断：                                              │
│    ├─ "帮我找XX行业的商机线索" → 触发工作流1（线索开发）│
│    └─ "帮我做XX客户的价值营销方案" → 触发工作流2（价值营销）│
│                                                          │
│  自动识别 + 自动路由 + 自动执行                          │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**核心价值**：
- 不是"两个独立专家"，而是"一个专家，两个工作流"
- 不是"用户要区分"，而是"系统自动识别意图"
- 不是"功能叠加"，而是"从线索到价值闭环"

## 01 工作流1：线索开发（找客户）

### 适用场景

```
触发条件（满足任意一个）：
  1. 用户提到"行业 + 商机线索"（如："帮我找汽配行业的商机线索"）
  2. 用户提到"客户 + 营销邮件"（如："帮我给均胜电子写营销邮件"）
  3. 用户提到"线索开发 + 关键人"（如："帮我识别牧原股份的关键决策人"）
```

### 线索开发自动生成流程

```
用户输入："我是[公司]，帮我开发[区域][行业]的商机线索"
    ↓
第1步：客户清单生成
  ├─ 基于区域产业数据
  ├─ 基于行业匹配度
  └─ 生成目标客户清单（TOP 50）
    ↓
第2步：商机识别
  ├─ 对每个客户进行商机识别
  ├─ 基于行业KPI痛点
  └─ 生成潜在商机清单（按优先级排序）
    ↓
第3步：关键人排序
  ├─ 识别每个客户的决策角色（EB/T-CB/T-AB/UB/EK）
  ├─ 对每个角色进行态度/支持度分析
  └─ 生成关键人地图（谁是关键人？谁支持？谁反对？）
    ↓
第4步：营销邮件生成
  ├─ 基于客户痛点+商机+关键人
  ├─ 生成定制化营销邮件（不是群发模板）
  ├─ 包含：价值主张+案例引用+邀约话术
  └─ 输出：营销邮件（每客户1封，可批量生成）
    ↓
第5步：输出交付物
  ├─ 目标客户清单（Excel，含商机+关键人+营销邮件）
  ├─ 商机识别报告（Word）
  ├─ 关键人地图（可视化）
  └─ 营销邮件包（Word/Outlook格式）
```

## 02 工作流2：价值营销（卖价值）

### 适用场景

```
触发条件（满足任意一个）：
  1. 用户提到"客户 + 价值营销"（如："帮我做牧原股份的价值营销方案"）
  2. 用户提到"价值主张 + ROI"（如："帮我计算XX客户的价值主张和ROI"）
  3. 用户提到"需求 + 岗位 + 指标"（如："帮我梳理XX客户的岗位KPI"）
```

### 价值营销五步法（核心方法论）

```
第一步：客户有什么需求？
   ↓   （基于客户洞察）
   ├─ 显性需求：客户主动表达的痛点
   └─ 隐性需求：客户未意识到但存在的痛点
   
第二步：哪些岗位涉及？他们的KPI是什么？
   ↓   （角色-指标映射）
   ├─ 决策角色识别（EB/T-CB/T-AB/UB/EK）
   ├─ 每个角色的KPI提取（行业自适应）
   └─ KPI baseline与target梳理
   
第三步：我们有什么产品/能力？
   ↓   （能力匹配）
   ├─ 产品功能清单
   ├─ 行业化解决方案
   └─ 差异化优势
   
第四步：给每个岗位创造什么价值？
   ↓   （价值主张）
   ├─ 每个KPI对应的价值点
   ├─ 价值点量化（提升X%/降低Y%/缩短Z%）
   └─ 差异化价值主张卡
   
第五步：价值如何量化？
   ↓   （数据驱动）
   ├─ ROI计算公式
   ├─ 3年收益测算
   ├─ 回本周期计算
   └─ 输出：价值营销方案 + ROI分析
```

## 03 价值营销自动生成流程

```
用户输入："我是[公司]，帮我做[客户名称]的价值营销方案"
    ↓
第1步：客户认知
  ├─ 检索客户洞察（已完成的360度洞察）
  ├─ 提取客户需求清单（显性+隐性）
  └─ 确认客户行业与业务特征
    ↓
第2步：角色-KPI映射
  ├─ 识别决策角色（EB/T-CB/T-AB/UB/EK）
  ├─ 提取每个角色的KPI（行业自适应）
  ├─ 梳理KPI baseline与target
  └─ 输出：角色-KPI矩阵
    ↓
第3步：能力匹配
  ├─ 检索产品/能力清单
  ├─ 匹配客户需求与产品功能
  ├─ 识别差异化优势
  └─ 输出：能力匹配矩阵
    ↓
第4步：价值主张生成
  ├─ 每个KPI对应的价值点
  ├─ 价值点量化（提升X%/降低Y%）
  ├─ 生成差异化价值主张卡
  └─ 输出：价值主张卡（每角色1张）
    ↓
第5步：ROI量化
  ├─ 选择ROI计算公式（基于场景）
  ├─ 输入参数（投资额/年收益/成本节约）
  ├─ 计算3年ROI与回本周期
  └─ 输出：ROI分析报告
    ↓
第6步：方案整合
  ├─ 整合所有价值主张卡
  ├─ 整合ROI分析
  ├─ 生成价值营销方案（Word/PDF）
  └─ 输出：价值营销方案 + ROI分析 + 价值主张卡
   
耗时：约60秒
```

#### 第五步：ROI分析

**投资测算**（牧原股份）：

| 投资项目 | 投资额（万） | 年收益（万） | 回本周期 |
|:--------|:--------------|:--------------|:----------|
| 智能化养殖云 | 500 | 3000（PSY提升） | 2个月 |
| 供应链管理云 | 300 | 1200（物流成本降低） | 3个月 |
| 财务云 | 200 | 500（报表效率提升） | 5个月 |
| PaaS平台（集成） | 400 | 800（数据打通） | 6个月 |
| **总计** | **1400万** | **5500万/年** | **3个月** |

**3年ROI计算**：
```
总投资：1400万
年收益：5500万
3年总收益：5500万 × 3年 = 16500万
3年净利润：16500万 - 1400万 = 15100万
3年ROI：(15100万 / 1400万) × 100% = 1079%
回本周期：3个月
```

**价值营销方案结论**：
- 推荐方案：一体化平台（智能化养殖云 + 供应链管理云 + 财务云 + PaaS平台）
- 3年ROI：1079%
- 回本周期：3个月
- 差异化优势：行业化深耕 + 快速上线 + 成本优势

## 07 质量标准

### 线索开发质量标准

| 维度 | 标准 | 检查方法 |
|:----|:------|:--------|
| **完整性** | TOP 20客户全部完成商机识别+关键人排序 | 检查客户清单 |
| **定制化** | 每客户营销邮件不同（不是群发模板） | 检查营销邮件包 |
| **可执行性** | 关键人地图具体到"角色+态度+支持度" | 检查关键人地图 |
| **数据准确性** | 客户信息/商机/关键人有依据 | 检查数据来源 |

### 价值营销质量标准

| 维度 | 标准 | 检查方法 |
|:----|:------|:--------|
| **完整性** | 五步法全部完成 | 检查价值营销方案 |
| **量化** | 每个价值点都有量化数据（提升X%/降低Y%） | 检查价值主张卡 |
| **ROI准确** | ROI计算公式有明确参数 | 检查ROI分析报告 |
| **差异化** | 每个客户的价值主张不同 | 检查价值主张卡 |
