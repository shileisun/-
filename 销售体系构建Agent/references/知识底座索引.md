## 关键方法论溯源

### 崔建中（价值型销售创立者）
- 业绩公式：销售业绩 = (商机数量 × 转化率 × 客单价) / 销售周期
- 三层结构：业绩层 → 策略层 → 活动层
- 执行四步法：简化目标 → 目标转会行为 → 实施记分表 → 建立问责制
- 核心思想：管理本身不能提升绩效，除非作用于销售行为

### C139模型
- C = Coach教练评分（13个指标）
- 1 = Win决定力指标（最高决策者态度）
- 3 = First趋赢力标杆（决策层评价）
- 9 = Clear必清事项（项目形势把握）
- 胜负拐点：0W2F5C=死亡拐点，1W1F5C=胜利拐点

### 标准化体系三层架构
- 通用层（销售方法论）
- 行业层（行业专业团队）
- 直销全流程层 + 代理商复制层
- 支撑保障层
- 80/20原则：80%赋能 + 20%管理抓手
