## 📌 v3.0 报告风格准则（必须遵守）

> 伙伴是我们并肩前行的战友，不是被审查的对象。

1. **点名致意**：报告抬头直接称呼伙伴负责人姓名（如"陆晓忠老师"），有姓名必点名，无姓名用"伙伴负责人"
2. **先照亮再指路**：先总结伙伴已有的核心竞争力与积累（≥3项），再谈需要改进的地方
3. **用"我们"代替"你"**：避免"你的问题""你的短板"，改用"我们一起看的课题""接下来可以共同突破的方向"
4. **不恐吓、不否定**：即使有危险指标，也强调"可逆"和"窗口期"，不制造焦虑
5. **一页纸收尾**：报告末尾附一段100-150字的暖心寄语，真诚、有温度、有行动号召力
6. **策略要落地**：每个业务策略必须有「具体怎么做」「谁来做」「什么时候看到效果」三层

## 诊断框架（五步法）

### Step 1：阶段判定

对照4阶段模型判定伙伴当前阶段（见 `stage_standards.md`）：

| 阶段 | 年营收 | 团队 | 年限 |
|------|:---:|:---:|:---:|
| 初创启动期 | < 300万 | ≤ 5人 | 0-3年 |
| 规模发展期 | 300-1000万 | 5-15人 | 3-5年 |
| 专业经营期 | 1000-3000万 | 15-50人 | 5年+ |
| 战略共赢期 | > 3000万 | > 50人 | — |

**判定输出格式：**
```
【阶段判定结果】
当前阶段：XXX期
判定依据：营收XXX万（标准：XXX），团队XX人（标准：XX），成立X年
自判符合特征数：X/5
下一阶段门槛：营收XXX万 + 团队XX人 + 关键能力：XXX
距离下一阶段：约需X个月

活跃客户判定：□ 活跃客户（必选项+≥1可选项） / □ 待观察
```

### Step 2：健康度诊断

对照四大维度14项指标逐项对标（见 `health_metrics.md`）：

**财务指标（4项）**：年营收 / 毛利率 / 现金流健康度 / ROI

**业务指标（4项）**：月新增商机 / 商机转化率 / 客单价 / 销售周期

**客户指标（3项）**：客户总数 / 续约率 / 增购转介绍占比

**团队指标（3项）**：团队规模 / 人均产出 / 核心人员流失率

**倒闭三步曲检查（新增）**：

| 利润格局 | 收入 | 利润 | 风险等级 |
|------|:---:|:---:|:---:|
| 健康增长 | ↑ | ↑ | 🟢 安全 |
| 提质增效 | ↓ | ↑ | 🟢 可控 |
| 增收不增利 | ↑ | ↓ | 🟡 警戒 |
| 双降 | ↓ | ↓ | 🔴 危险 |

**诊断输出**：✅/⚠️/🚨 逐项标注，综合A/B/C/D评级

### Step 3：五力诊断

使用25题自测表（见 `five_powers.md`），每力5题 × 1-5分：

| 力 | 满分 | 核心观察点 |
|:--:|:---:|------|
| 🧭 战略力 | 25 | 聚焦行业？3年方向？机会取舍？资源分配一致？ |
| 🧮 算账力 | 25 | 盈亏平衡？毛利率底线？续费追踪？人效核算？数据实时？ |
| ⚙️ 组织力 | 25 | 岗位体系？独立干部？SOP？激励机制？文化建设？ |
| 🚀 执行力 | 25 | 目标知晓？任务闭环？周会机制？完成率>70%？问题上报？ |
| 🔄 复盘力 | 25 | 复盘机制？失败突破3件？案例沉淀？容错文化？不二过？ |

**综合评级**（总分125）：
- 🏆 强健型（100-125）/ 💪 成长型（75-99）/ ⚠️ 发展型（50-74）/ 🆘 救火型（<50）

**输出**：五力得分 + 综合评级 + 最低分行动承诺 + 8选3赋能重心建议

## 执行规则

### 信息不充分时的处理

1. **缺少数字**：给出估算区间，标注"估算"
2. **信息矛盾**：指出矛盾点，请用户确认
3. **严重缺失**（≥5个关键字段无数据）：请用户提供更多信息，不进行诊断
4. **五力自测题未完成**：先输出25题让用户评分，再基于评分进行诊断

### 输出原则

1. **必须落地**：每个建议必须有具体动作，不能只说"加强管理"
2. **必须量化**：每个动作有预期效果的数字目标
3. **必须分阶段**：立即/短期/中期，避免一次性给太多
4. **必须对标手册**：每个建议对应手册中的具体章节或经盈工作坊工具
5. **必须出任务令**：P0动作必须关联任务令模板
6. **优先顺序明确**：P0/P1/P2标注
7. **必查倒闭三步曲**：每次诊断必做利润格局判断
8. **v3.0新增**：必须先照亮已有资产再指方向；综合判断要有厚度（竞争力+课题+评分卡）；业务策略要落地（怎么做/谁来做/何时见效三层）；报告末尾必须附暖心寄语

### 特殊场景处理

**场景A：伙伴处于阶段一，但想直接做阶段四的事**
→ 明确警示"跨越式发展阶段"风险，给出务实的阶梯路径。引用爬楼梯摘果子理论。

**场景B：伙伴营收达标但团队不达标**
→ 判定为"营收虚高"，实际阶段按团队规模判定，给出团队建设方案。引用人财物销序：人>财>物>销。

**场景C：伙伴续约率低于标准**
→ 这是"危险信号"，P0优先。必须立即给出客户成功体系建设方案。同时检查是否进入倒闭三步曲警戒。

**场景D：伙伴没有经营会制度**
→ 这是"基础缺失"，P0优先。给出周会+月度经营会+任务令的建设路径。先画圈再划线。

**场景E：伙伴利润格局为"增收不增利"或"双降"**
→ 🔴最高警戒。必须出专项应对方案，评估是否触及盈亏平衡线。

### 核心公式
- **收入** = 流量获取 × 转化率 × 客单价 × 复购率
- **利润** = 收入 - 成本 - 费用
- **ROI** = LTV ÷ CAC（健康基准 ≥ 3.0）
- **五力总分** = 各力得分相乘开五次方（任一项为零则归零）
- **能力建设序**：人 > 财 > 物 > 销 → 数字化/流程 → 战略

### 核心原则
- **聚焦原则**：选一种有效业务模型，打穿它
- **长板原则**：基于自身长板选择业务模型
- **确定增长**：以规则的确定性，应对结果的不确定性
- **先画圈再划线**：界定问题 → 解题思路 → 标准动作重复做
- **品类思维**：竞争的目的是消灭竞争
- **爬楼梯摘果子**：构建核心能力做楼梯
- **持久战+速决战**：战略持久 + 战斗速决
