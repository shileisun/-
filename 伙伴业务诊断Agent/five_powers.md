# 经营力五力模型（经盈工作坊版）

> 📌 **来源**：《纷享销客伙伴盈利手册3.0》+ 经盈工作坊第一期工具①五力自测表
> 📌 每力5题，每题1-5分，满分25分/力，总分125分

---

## 五力比喻

| 力 | 比喻 | 核心命题 |
|:--:|------|---------|
| 🧭 战略力 | 方向盘 | 选对方向，知道要去哪里 |
| 🧮 算账力 | 油量表 | 算清盈亏，知道还能走多远 |
| ⚙️ 组织力 | 传动系统 | 搭好发动机，让团队能转起来 |
| 🚀 执行力 | 车轮 | 从计划到结果，把事做成 |
| 🔄 复盘力 | 后视镜 | 在错误中进化，不犯同样的错 |

**木桶效应**：五力均分 = 各力得分相乘开五次方。任一项归零，整体归零。

## 🧮 第二力：算账力（算清盈亏）

| # | 自测题目 | 得分 |
|---|---------|:---:|
| 1 | 我能说出公司每月的盈亏平衡点是多少 | /5 |
| 2 | 我清楚每一单的毛利率，并有底线标准 | /5 |
| 3 | 我有完整的续费追踪机制，每月知道续费率 | /5 |
| 4 | 我能按人/团队核算每个人的产出贡献 | /5 |
| 5 | 我的财务数据是实时可见的，不靠月底对账 | /5 |

**算账力合计**：___/25

**诊断观察点**（来自工具③）：
- □ 知道盈亏平衡点
- □ 有无续费追踪机制
- □ 能否按人核算产出

### 算账力经济学（来自手册）

| 指标 | 公式 | 健康基准 |
|------|------|---------|
| LTV | ARR × 续约年数 | ≥ 3×CAC |
| CAC | 总费用 ÷ 新客数 | 各阶段递增 |
| ROI | LTV ÷ CAC | ≥ 3.0 |
| 盈亏平衡点 | 固定成本 ÷ 毛利率 | 每月追踪 |

## ⚙️ 第三力：组织力（搭好发动机）

| # | 自测题目 | 得分 |
|---|---------|:---:|
| 1 | 公司有清晰的岗位职责和考核标准，不靠老板盯 | /5 |
| 2 | 公司有能独立带团队的干部，不是所有事都靠我 | /5 |
| 3 | 公司的核心业务流程有文档/SOP，新人可以上手 | /5 |
| 4 | 团队有明确的激励机制，优秀员工愿意留下 | /5 |
| 5 | 公司文化是可感知的，员工行为与文化一致 | /5 |

**组织力合计**：___/25

**诊断观察点**（来自工具③）：
- □ 有无独立带团队的干部
- □ 有无SOP/岗位体系
- □ 团队留存率如何

### 人财物销序（来自可视化经营分析会）

```
人 > 财 > 物 > 销

放量前必须回答：有没有优秀的团队？有没有优秀的干部？
→ 没有就不要搞规模

初始阶段：能力因创始人而异
规模阶段：人大于财，财大于物，物大于销
```

**招人假设**：①能满足当前岗位要求 ②为未来的填补做准备。招高潜人才，不"一个萝卜一个坑"。

## 🚀 第四力：执行力（从计划到结果）

| # | 自测题目 | 得分 |
|---|---------|:---:|
| 1 | 公司有明确的季度/月度目标，且全员知晓 | /5 |
| 2 | 重要任务都有负责人、时间节点和完成标准 | /5 |
| 3 | 我每周有固定的业务回顾会，跟进执行进度 | /5 |
| 4 | 年初定的计划，到年底执行完成率超过70% | /5 |
| 5 | 团队遇到困难时有问题上报机制，不是靠等老板发现 | /5 |

**执行力合计**：___/25

**诊断观察点**（来自工具③）：
- □ 有无固定的周/月例会
- □ 目标完成率情况（>70%为健康）
- □ 任务是否有闭环机制

## 🔄 第五力：复盘力（在错误中进化）

| # | 自测题目 | 得分 |
|---|---------|:---:|
| 1 | 公司有固定的月度/季度复盘机制，不只开庆功会 | /5 |

| 4 | 我鼓励团队汇报问题，不会因为汇报坏消息而被批评 | /5 |
| 5 | 同样的问题在我们公司不会犯第二次 | /5 |

**复盘力合计**：___/25

**诊断观察点**（来自工具③）：
- □ 有无定期复盘机制
- □ 同样的问题会否反复出现
- □ 有无案例沉淀习惯

## 综合评级（满分125）

| 总分 | 评级 | 行动建议 |
|:---:|:---:|------|
| 100-125 | 🏆 **强健型** | 维持优势，重点打磨最短板那一力 |
| 75-99 | 💪 **成长型** | 找到最低分那一力，制定专项改进计划 |
| 50-74 | ⚠️ **发展型** | 排优先级，从最影响生存的那一力开始 |
| < 50 | 🆘 **救火型** | 先聚焦活下来，战略力和算账力是底线 |

**五力雷达图解读**：
- 均 > 4分：全面领先，维持即可
- 某力 ≤ 2分：**木桶效应触发**，该力即为当前最大瓶颈
- 战略力 + 算账力同时 < 3分：生存危机信号，P0优先
