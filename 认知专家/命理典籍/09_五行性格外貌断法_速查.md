## 一、五行性格速查

### 1.1 十日干性格画像

**甲木（参天大树）**
- 关键词：仁、直、冲
- 性格：正直有担当，领导力强，宁折不弯。表面刚强内心柔软，不服输但有底线。
- 致命弱点：一根筋到底，不撞南墙不回头
- 像谁：公司里那个死扛项目不放手的技术骨干

**乙木（花草藤蔓）**
- 关键词：柔、韧、缠
- 性格：灵活善变，适应力极强，柔中带刚。善于借力打力，懂得审时度势。
- 致命弱点：优柔寡断，依赖性太强
- 像谁：那个谁都不得罪、谁都能搭上话的社交达人

**丙火（太阳高照）**
- 关键词：炎、亮、散
- 性格：热情奔放，慷慨大方，天生领导气质。感染力强，走到哪亮到哪。
- 致命弱点：三分钟热度，好面子到死
- 像谁：公司里那个一开会就嗨、讲完就忘的领导

**丁火（烛火灯芯）**
- 关键词：烛、思、执
- 性格：洞察力极强，心思细腻，重感情。表面温和内心执着，一旦认定九头牛拉不回。
- 致命弱点：钻牛角尖，敏感多疑
- 像谁：那个什么都看在眼里、但什么都不说的同事

**戊土（高山大地）**
- 关键词：稳、厚、守
- 性格：厚重稳健，诚信可靠，能扛大事。包容力强，不轻易发火，但一旦发火山崩地裂。
- 致命弱点：太迟钝，不懂转弯
- 像谁：那个什么活都能干、但永远不主动的老实人

**己土（田野沃土）**
- 关键词：润、让、忍
- 性格：谦逊细心，服务意识强，善于照顾人。默默付出，不求回报。
- 致命弱点：没主见，容易委屈自己
- 像谁：那个总帮大家订外卖、但从不点自己喜欢的行政

**庚金（顽铁刀剑）**
- 关键词：刚、义、断
- 性格：刚毅果断，讲义气，执行力强。做事干脆利落，不喜欢拖泥带水。
- 致命弱点：太过刚硬，不懂退让，容易伤人伤己
- 像谁：那个敢跟老板硬刚、但朋友有难第一个冲上去的兄弟

**辛金（珠玉首饰）**
- 关键词：秀、精、娇
- 性格：精致敏感，审美极高，讲品质讲格调。重面子，追求完美。
- 致命弱点：太脆弱，经不起打击，矫情
- 像谁：那个穿搭永远最精致、但被批评一句就难受三天的人

**壬水（大江大河）**
- 关键词：智、变、漂
- 性格：智慧通达，变通力强，格局大。不受约束，追求自由。
- 致命弱点：太散漫，飘忽不定，三心二意
- 像谁：那个总说"我有个想法"、但过两天又有新想法的人

**癸水（雨露泉水）**
- 关键词：润、察、忧
- 性格：细腻敏感，直觉力极强，善于观察。润物无声，默默付出。
- 致命弱点：太内耗，多想，容易消沉
- 像谁：那个总是默默把事做了、但心里翻江倒海的人

## 二、五行外貌速查

### 2.1 五行定体型口诀

```
木瘦金方水主肥，土形敦厚背如龟。
上尖下阔名为火，语浅言深仔细推。
```

### 2.2 十日干外貌口诀（盲派秘传）

**甲木**：面长额宽骨格清，眉浓眼正发粗硬。为人正直不弯腰，身材高大如松柏。

**乙木**：眉清目秀面如鹅蛋，发细指纤体态轻盈。温柔似水善交际，但见风使舵没定力。

**丙火**：面红额亮眼如炬，发粗体壮气宇轩昂。走到哪里都显眼，但火气一上来就炸。

**丁火**：面尖额窄眼细长，发细唇薄心思深。表面温和内藏着刀，看人看事一针见血。

**戊土**：方脸鼻大唇厚实，背厚腰圆步态稳。看似木讷实则深沉，开口就是金句。

**己土**：圆脸肉多眼下垂，发柔肤细态和蔼。看着老实实际精明，做人做事滴水不漏。

**庚金**：面方颧高眼如刀，发硬面棱不怒自威。做事杀伐决断不留情，但嘴硬心软。

**辛金**：五官精致肤似雪，丹凤眼挑发细亮。看着柔弱实则心高气傲，经不起委屈。

**壬水**：圆脸宽下巴大眼，体毛多发量多。胖瘦都能长，但下半身偏壮。走一步看三步。

**癸水**：面黑瘦五官紧凑，小眼深邃发细少。看着不起眼实则深不可测，城府极深。

## 三、六亲关系速查

### 3.1 六亲星与宫位双看法

> 核心原则：**星宫同看，星为用、宫为体**
> - 星旺宫稳 → 六亲关系好
> - 星旺宫破 → 六亲有能力但关系不稳
> - 星弱宫稳 → 六亲关系还行但对方能力一般
> - 星弱宫破 → 六亲缘薄

### 3.2 配偶关系速判表

| 日支十神 | 配偶性格 | 婚姻质量 | 关键提示 |
|----------|----------|----------|----------|
| 正财（男）/正官（女） | 贤良稳重 | ★★★★★ | 最佳配置 |
| 偏财（男）/七杀（女） | 能干但强势 | ★★★☆☆ | 有波折 |
| 食神 | 温柔有才华 | ★★★★☆ | 享受型 |
| 伤官 | 叛逆意见多 | ★★☆☆☆ | 易争吵 |
| 正印 | 照顾型像长辈 | ★★★★☆ | 缺激情 |
| 偏印 | 难以捉摸 | ★★★☆☆ | 忽冷忽热 |
| 比肩 | 兄弟朋友型 | ★★★☆☆ | 太熟 |
| 劫财 | 争抢型 | ★★☆☆☆ | 易冲突 |
| 七杀 | 强势管控型 | ★★☆☆☆ | 压力大 |

### 3.3 盲派看父母谁先亡口诀

```
若问父母谁先亡，月令年柱细审详。
年干为父支为母，五行生克记心上。
月令克谁谁先死，相生相比寿命长。
```

### 3.4 盲派六亲活变三步法

**第一步**：定六亲星（找十神）
**第二步**：看六亲星与日主关系（生/克/比/泄/耗）
**第三步**：把六亲星当"日干"，重新定十神，看其他干支对六亲星的生克

> 这个方法能断出六亲的具体情况，而不只是"关系好不好"。
> 例如：断母亲 → 正印被偏财克 → 母亲被父亲压制 → 父亲强势、母亲弱势

> **使用方式**：断命时，先查日干性格画像 → 再查月令加成 → 再查十神组合 → 再查外貌 → 再查六亲 → 综合输出。
