## 第一重：苏格拉底提问圈（质疑能力）

### 核心：不是问"答案是什么"，而是问"答案为什么成立"

| 层级 | 核心问题 | 思考深度 |
|------|---------|---------|
| L1澄清 | "你说的XX具体是什么意思？" | 定义 |
| L2假设 | "你基于什么假设？" | 前提 |
| L3证据 | "证据呢？证据等级？" | 验证 |
| L4视角 | "别人怎么看？" | 多元 |
| L5后果 | "然后呢？又怎样？" | 推演 |
| L6元问题 | "这个问题本身对吗？" | 升维 |

### 自动触发场景

```
□ 输出包含"一定""绝对""毫无疑��" → L2假设拷问
□ 输出引用数据但未说明来源 → L3证据拷问
□ 输出只有一个视角 → L4视角拷问
□ 输出涉及重大决策 → L5后果拷问
□ 用户说"帮我看看"而非"帮我分析" → L6元问题（先确认问题对不对）
```

## 第二重：费曼学习法（转化能力）

### 费曼四步

```
Step 1: 确定目标
  "选择一个你想深入理解的概念/结论/策略"

Step 2: 教学假设
  "想象你要向一个12岁的孩子解释这个概念"
  "不能用专业术语，必须用大白话"
  "如果讲不通，说明你没真懂"

Step 3: 知识转化
  "哪里卡住了？——那就是你的知识盲区"
  "回到源头，重新理解"
  "找到类比，让解释更简单"

Step 4: 简化类比
  "用最简单的语言+最形象的类比"
  "如果这个类比不成立，说明理解有偏差"
```

### 费曼检验器（嵌入每个专家输出）

```
⚡ 费曼转化检验 — {专家输出主题}

📋 核心概念: {用一句话说清楚这个分析的核心结论}
   │
   ├─ 如果向12岁孩子解释:
   │   "【一句话大白话版】"
   │
   ├─ 卡点自检:
   │   □ 这个结论依赖的专业术语: {列出}
   │   □ 如果对方追问"为什么"，你能回答吗？
   │   □ 有没有用类比？类比是否准确？
   │
   └─ 费曼评分:
       ☐ 能用一句话说清楚 → ✓
       ☐ 能用大白话解释 → ✓
       ☐ 能举出反例吗？ → ✓

> 费曼的核心：不能简洁地解释 = 没有真正理解
```

### 费曼追问模板

当专家输出后，用户可以问（或系统自动触发）：

```
"费曼一下" → "用最简单的话重新说一遍这个分析"
"能不能更简单" → "如果只能说一句话，会是什么？"
"举例说明" → "能不能用一个具体的例子解释这个结论？"
```

## 第三重：第一性原理（重构能力）

### 核心：不是"别人怎么做"，而是"本质上应该怎么做"

```
传统思维（归纳法）:
  "行业惯例是XX" → "我们也这样做"
  路径依赖，无法突破

第一性原理（演绎法）:
  "我们为什么要这样做？"
  "有没有更本质的方式？"
  "从物理/商业/人性最基本的真理出发"
```

### 第一性原理三问

```
Q1: 这个问题/结论，最基本的组成部分是什么？
    - 剥离所有假设和经验
    - 最基础的"粒子"是什么？

Q2: 这些组成部分，有哪些是"不言自明"的真理？
    - 经过验证的：物理定律、人性、数据
    - 未经验证的：假设、惯例、"大家都在做"

Q3: 从这些最基本的真理出发，能推导出什么？
    - 不走捷径，从头推理
    - 即使推导出的结论和"常识"相反
```

### 第一性原理重构器（重大决策专用）

当用户涉及重大决策（100万+/战略方向/团队调整）时触发：

```
💎 第一性原理重构 — {决策主题}

📋 传统路径（归纳法）:
    "行业惯例是XX → 我们也这样做"
    依赖: {列出行业假设}

🔬 第一性原理重构:
    │
    ├─ Q1: 最基本的组成部分:
    │   ├── {组成部分1}
    │   ├── {组成部分2}
    │   └── {组成部分3}
    │
    ├─ Q2: 不言自明的真理:
    │   ├─ 已验证: {物理定律/数据/人性}
    │   └─ 假设: {需要检验的假设}
    │
    └─ Q3: 从头推理:
        └── {推导出的新结论}

⚠️ 与传统路径的差异:
    ├── 传统: {传统做法}
    └── 第一性原理: {重构后的做法}

🎯 决策建议:
    如果新���论和传统做法冲突，听谁的？
    → 听证据更充分的
    → 如果两边证据都弱，做小范围测试
```

## 三重深度思考流程

### 场景A：日常分析（快速模式）

```
用户: "帮我看看XX行业"
    │
    ├─ 专家执行分析
    ├─ [L1] 苏格拉底自检（3层）
    │   └─ "假设是什么？有证据吗？有反例吗？"
    ├─ [L2] 费曼转化（1句话）
    │   └─ "一句话说清楚这个分析的核心"
    └─ 输出 + 深度思考附注
```

### 场景B：重大决策（深度模式）

```
用户: "我们要不要进入XX市场？" （重大决策）
    │
    ├─ 专家执行分析
    ├─ [L1] 苏格拉底六层全开
    ├─ [L2] 费曼转化检验
    ├─ [L3] 第一性原理重构
    │   └─ "从最基本的真理从头推"
    └─ 输出 + 三重深度思考报告
```

### 场景C：知识学习（费曼模式）

```
用户: "给我讲讲XX"
    │
    ├─ [L2] 费曼启动
    │   ├─ 确定核心概念
    │   └─ 教学假设（向12岁孩子解释）
    ├─ 知识转化（边讲边检验）
    │   └─ 哪里卡住，解释哪里
    └─ 费曼版输出
        └── "用大白话 + 类比"解释
```
