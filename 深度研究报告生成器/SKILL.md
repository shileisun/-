# 深度研究报告生成器

## 目的

生成深度行业研究分析报告，超越简单信息汇总，揭示产业链深层逻辑和结构性变化，支持战略决策和投资判断。

## 使用场景

- 行业白皮书生成
- 政策影响深度分析
- 技术趋势研究
- IPO热潮分析
- 产业链结构性变化研究
- 投资机会与风险评估

## CO-STAR深度分析框架

### C (Context) - 背景设定

提供宏观背景：
```
以[年份][身份]身份，针对[时间段]间[行业]企业[现象]进行深度分析。需综合考量：[关键背景要素1]、[关键背景要素2]等。
```

### O (Objective) - 核心目标

分解分析目标：
```
**宏观维度目标** → [描述，如：行业全景、政策红利、风险预警]
**微观维度目标** → [描述，如：产业链结构性变化、技术突破点]
**预测维度目标** → [描述，如：未来趋势推演]
```

### S (Style) - 输出风格

双风格输出：
- **白皮书部分**：采用第三方行业报告语言（如头豹研究院/灼识咨询体例）
- **研究部分**：采用学术研究视角（引用德勤/麦肯锡行业模型）

### T (Task) - 具体任务

#### 1. 宏观维度

- 统计[时间段]行业[指标]趋势（分[维度1/维度2/维度3/维度4]四大板块）
- 绘制[因素A]与[因素B]相关性图谱

#### 2. 微观维度

- 选取典型案例进行SWOT-CL（资本杠杆）分析
- 建立[变量X]与[变量Y]回归模型

#### 3. 预测维度

- 使用PESTEL框架推演[时间段]趋势
- 情景模拟分析

### A (Audience) - 受众需求

明确受众：
```
1. **客户需求**：[描述，如：投资人关注财务回报周期]
2. **自我需求**：[描述，如：建立XX评估指标体系]
```

### R (Response) - 格式要求

生成结构严谨的双输出文档：

**Part 1 行业白皮书框架**
```
▷ 封面标题：《[标题]》
▷ 核心目录：[概览→动态→解码→地图→建议]
```

**Part 2 深度研究附录**
```
▷ 含原始数据集
▷ 含关键技术曲线
▷ 需标注未公开访谈信息源用[✱]符号代替
```

## 关键分析方法

### 矛盾点挖掘
- 识别行业内部矛盾（如：融资过热 vs 量产能力滞后）
- 分析矛盾产生的深层原因

### 颠覆性结论
- 揭示反直觉的发现
- 提供数据支撑和逻辑推理

### 根因分析
- 使用"5 Why"分析法追溯根本原因
- 避免表面结论

### 动态视角
- 进行时间序列分析（3-5年历史数据）
- 识别拐点和持续性趋势
- 对比企业对相同外部事件的反应

## 数据可视化建议

报告中可包含：
- IPO募资规模Top10领域分布雷达图
- 政策支持与IPO过会率相关性图谱
- 产业链环节对比表格
- 趋势预测曲线图

## 注意事项

1. 数据驱动：优先使用量化指标，辅以定性分析
2. 平衡对比：客观呈现优劣势
3. 决策导向：聚焦对指定决策场景的应用
4. 标注数据来源，确保可追溯性
5. 明确标注假设条件和不确定性
