## 〇、总论：为什么"八字自带自问"

传统自我认知靠"问答"——问自己、问教练、问朋友。但八字不需要问，因为：

| 自我认知问题 | 八字读取位置 | 不用问，直接读 |
|-------------|-------------|---------------|
| 我是谁？ | 日干+日支 | 你的本质身份 |

| **戊土** | 承载者 | 一个必须扛事的人 | 大山——不扛就不是山 |
| **己土** | 滋养者 | 一个必须照顾人的人 | 沃土——不付出就不安 |

## 八、八字版"每日三问" — 从盘面直接读取

> 原版每日三问需要你自我反思。八字版直接从**流日干支**读取答案。

### 8.1 流日读取法

| 问题 | 读取位置 | 判断方法 |
|------|---------|---------|
| 今天哪件事让我感到最有能量？ | 流日天干是否=你的**用神/喜神** | 是→今天做难事/做决定；否→今天做轻松事 |
| 今天哪件事让我感到最消耗？ | 流日天干是否=你的**忌神** | 是→今天少做决定/少社交；否→可以正常活动 |
| 今天我是否做了一个更接近"真实自我"的选择？ | 流日地支与**日支**的关系 | 生/比→是；冲/克→被干扰了 |

### 8.2 流日五行与能量管理

| 流日五行 | 今日能量类型 | 适合做什么 | 不适合做什么 |
|---------|-----------|----------|-----------|
| 甲乙日 | 生长日 | 学习、谈合作、启动新项目 | 做终局决定 |
| 丙丁日 | 表达日 | 演讲、谈判、社交、提案 | 独自闷头干活 |
| 戊己日 | 稳定日 | 执行、交付、签合同 | 启动全新方向 |
| 庚辛日 | 决断日 | 做决定、砍掉不必要、清理 | 纠结犹豫 |
| 壬癸日 | 流动日 | 思考、策略、见人、收集信息 | 做硬性决定 |

### 8.3 流日与日主关系速查

| 流日与日主关系 | 今天的"自我状态" | 建议 |
|-------------|---------------|------|
| 比肩日 | 自信、有底气 | 适合做需要气场的事 |
| 劫财日 | 冲动、想争 | 不适合做金钱决定 |
| 食神日 | 愉悦、有创意 | 适合创作、社交 |
| 伤官日 | 叛逆、想表达 | 适合写东西，不适合跟上级沟通 |
| 偏财日 | 灵活、想花钱 | 适合社交，不适合大额消费 |
| 正财日 | 务实、想赚钱 | 适合签单、做商业决策 |
| 七杀日 | 有压力、想突破 | 适合攻克难题，注意身体 |
| 正官日 | 守规矩、想被认可 | 适合汇报、述职、正式场合 |
| 偏印日 | 想独处、想钻研 | 适合学习、研究，不适合社交 |
| 正印日 | 想学习、想被照顾 | 适合充电、找导师 |

## 九、整合输出规范 — "八字自我认知画像"

### 9.1 输出格式

```
┌─────────────────────────────────────────┐
│         「八字自我认知画像」              │
│                                         │
│  本质身份：[日干核心+日支内在]           │
│  充电桩：[用神+喜神，3个具体场景]        │
│  能量黑洞：[忌神+仇神，3个具体场景]      │
│  核心价值观：[格局定价值观]              │
│  关系模式：[日支+冲克]                  │
│  最深恐惧：[忌神恐惧]                   │
│  真正优势：[喜神优势]                   │
│  真正盲区：[忌神盲区]                   │
│  优势与劣势是同一件事：[喜忌同源]       │
│  当前人生阶段：[大运+流年]              │
│  给现在的你的建议：[一句话]             │
└─────────────────────────────────────────┘
```

### 9.2 锐评规范

- ✅ 必须有"优势与劣势是同一件事"的揭示——这是八字自我认知的精华
- ✅ 必须有"你一直回避但迟早要面对的事"——打破舒适区
- ✅ 必须有"你羡慕别人什么"的深层解读——不是虚荣，是未完成
- ✅ 锐评风格：麻辣、直白、有辣味，但不说"你有病"
- ❌ 不说"你需要改变"——只说"你如果不再害怕，你会……"
- ❌ 不说"你的问题是……"——只说"你的盲区是……"

## 十、与Step 6-9的衔接

| 模块 | 输出 | 自我认知模块取用 |
|------|------|---------------|
| Step 6 性格剖析 | 十日干性格+十神组合性格 | → 映射到"本质身份""优势盲区" |
| Step 7 外貌推断 | 五行外貌+日干面部 | → 映射到"本质身份"（你看起来像谁=你是谁的外显） |
| Step 8 六亲关系 | 父母/兄弟/配偶/子女关系 | → 映射到"关系模式""恐惧限制" |
| Step 9 天赋职业 | 十神天赋+格局事业 | → 映射到"价值观""优势盲区" |
| Step 10 自我认知 | 7维整合+每日三问 | → **吸收以上所有模块的输出，整合为"认识自己"的完整画像** |

**执行顺序**：先完成Step 6-9，再执行Step 10。Step 10是"收口"模块，把前面所有模块的输出整合成一幅"自我认知画像"。

---

> **核心理念**：八字不是告诉你"你会怎样"，而是告诉你"你是谁"。知道了自己是谁，该怎样自然就知道了。
