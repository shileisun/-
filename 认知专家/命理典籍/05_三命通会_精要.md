# 《三命通会》精要 · 万民英（明）

> 源出：明万民英（进士、进士第、翰林）。**集子平术之大成**，十二卷，篇幅最大、内容最全的命理百科全书。

## 一、地位
- "**命理学的百科全书**"——五本书里**内容最全**
- **集前人之大成**：收《渊海子平》《理愚歌》《玉照定真经》《子平渊源》等多家
- 几乎所有命理议题都**能找到出处**

## 三、核心思想

### 1. 命理学的"总原则"
> "**命者，天地之纲纪、阴阳之造化、五行之生克、八字之推排，皆不离乎理。**"

### 2. "五行之理"
- **相生**：木生火、火生土、土生金、金生水、水生木
- **相克**：木克土、土克水、水克火、火克金、金克木
- **相合**：甲己合、乙庚合、丙辛合、丁壬合、戊癸合
- **相冲**：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- **相刑**：寅巳申三刑、丑戌未三刑、子卯刑、辰午酉亥自刑
- **相害**：子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害
- **相破**：子酉破、丑辰破、寅亥破、卯午破、申巳破、未戌破
- **相墓**：辰为水土墓、戌为火墓、丑为金墓、未为木墓

### 3. 格局论的"集大成"
- 把格局分为**正格、变格、外格**
- 论格局**"成、败、破、救"** 四大条件
- 论**"相神"** 的辅助作用

### 4. 大运流年
- 大运为**十年之气**
- 流年为**当年之令**
- 太岁为**木神之主**

### 5. 神煞的"系统化"
三命通会**把神煞彻底系统化**：
- 把神煞按"吉"与"凶"分类
- 每个神煞有**查法、主事、化解**
- 是后世盲派神煞法的**标准范本**

### 6. 纳音、胎元、命宫、身宫
三命通会**详论胎元、命宫、身宫**：
- **胎元**：月干前一位为胎元之干、月支前一位为胎元之支
- **命宫**：以月支为起点，逆数至"卯"为命宫
- **身宫**：以月支为起点，顺数至"巳"为身宫

## 四、核心断语

1. "**命有格局、运有起伏，格局主命、运主时。**"
2. "**大运十年一换、流年一岁一应。**"
3. "**太岁为众煞之主、不可冲犯。**"
4. "**神煞可解不可抗，君子避之、小人抗之。**"
5. "**胎元主先天、命宫主后天、身宫主归宿。**"

## 五、三命通会的最大特色
- **"神煞系统"**——所有神煞**有查法、有主事、有化解**
- **"大运流年体系"**——最完整
- **"十二卷"** ——**命理学"工具书"**

## 六、实战要点

### 三命通会最常用的几卷
- **卷二**（格局）——断格局必看
- **卷三**（六亲）——断父母兄弟妻子必看
- **卷八**（流年）——断大运流年必看
- **卷十**（神煞）——查神煞用

### 三命通会的"案例卷"
- **卷十二**收历代名命案例——**现代命师仍可参考**

## 麻辣心得

三命通会像"**命理学大辞典**"——
- 你想知道"伤官见官"有哪些说法？→ 卷二、卷三
- 你想知道"驿马"怎么查、怎么断？→ 卷十
- 你想知道某年流年怎么断？→ 卷八
- 你想知道胎元命宫怎么算？→ 卷九

**断命时遇到拿不准的，**第一反应就是去翻三命通会**。

> **麻辣心得**：三命通会是"**集大成**"，所以不专精——专精要找**真诠（格局）、穷通（调候）、滴天髓（流通）、渊海（神煞）**。但讲"全"，三命通会第一。

## 七、五本书的关系

| 书 | 重点 | 角色 |
|---|---|---|
| **渊海子平** | 体系、神煞 | "**工具书**" |
| **子平真诠** | 格局成败 | "**格法心法**" |
| **穷通宝鉴** | 调候用神 | "**气候诊断**" |
| **滴天髓** | 流通、象法 | "**最高心法**" |
| **三命通会** | 集大成 | "**百科全书**" |

**断一个命，最优路线**：
1. **渊海** → 备齐十神、神煞、宫位工具
2. **真诠** → 定格局成败
3. **穷通** → 看调候、寒暖
4. **滴天髓** → 看流通、象法
5. **三命通会** → 查漏补缺、对照案例
6. **盲派** → 看宾主、体用、实战断事
