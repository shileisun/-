# 客户经营专家 v10.0

## 定位与价值

**客户经营 ≠ 销售管理**

```
┌─ 客户经营专家 ─────────────────────────────────┐
│                                                          │
│  客户经营体系（五步工作坊法）                      │
│    ├─ 第一步：确定经营目标                              │
│    ├─ 第二步：确定经营维度                              │
│    ├─ 第三步：制定经营策略                              │
│    ├─ 第四步：设计管理机制                            │
│    └─ 第五步：制定评估机制                            │
│                                                          │
│  + 三年战略经营计划                                    │
│    ├─ 第一年：建立信任（关系突破）                  │
│    ├─ 第二年：深化合作（价值呈现）                  │
│    ├─ 第三年：战略伙伴（共生共赢）                  │
│    └─ 逐年营收/回款目标                               │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**核心价值**：
- 不是"完成交易"，而是"客户经营"
- 不是"短期赢单"，而是"长期伙伴"
- 不是"销售视角"，而是"客户视角"（关注客户的客户成功）

## 01 客户经营体系（五步工作坊法）

### 第一步：确定经营目标

**经营目标层级体系**：

| 目标层级 | 经营目标 | 适用场景 |
|:---:|---------|---------|
| **战略层** | 构建战略关系 | 头部客户深度绑定，生态共建 |
| **标杆层** | 标杆声誉 | 行业灯塔客户，品牌背书 |
| **增长层** | 持续产出 / 增加客户价值 | 存量客户渗透率提升 |
| **基础层** | 保持续费 / 保持合作关系 | 老客户维护，防止流失 |
| **突破层** | 获得New Logo / 突破实现合作 | 新客户开发，竞品转换 |
| **生存层** | 形成收入 / 别被淘汰 | 初创期，现金流客户 |

**目标制定原则**：
- SMART原则（具体/可衡量/可达成/相关性/时限性）
- 与客户战略对齐
- 区分"必须达成"和"努力达成"

---

### 第二步：确定经营维度

**四大经营维度**：

```
维度1：客户分层分类（S/A/B/C四级）
  ├─ S级：战略客户（CEO/销售总监亲自经营）
  ├─ A级：关键客户（销售经理主导经营）
  ├─ B级：成长客户（销售代表定期跟进）
  └─ C级：基础客户（自动化营销+渠道覆盖）

维度2：关系发展阶段（五阶段）
  ├─ 陌生（0%信任）
  ├─ 认识（20%信任）
  ├─ 熟悉（40%信任）
  ├─ 信任（70%信任）
  └─ 同盟（90%信任）

维度3：业务渗透（多产品/多部门）
  ├─ 单产品 → 多产品
  ├─ 单部门 → 多部门
  ├─ 单场景 → 全流程
  └─ 供应商 → 战略伙伴

维度4：价值呈现（量化ROI）
  ├─ 效率提升（XX%）
  ├─ 成本降低（XX万/年）
  ├─ 收入增长（XX万/年）
  └─ 风险降低（XX%）
```

---

### 第三步：制定经营策略（八大经营要素）

| 要素 | 定义 | 示例 |
|:----|:------|:------|
| **1. 经营目标** | 为什么要经营这个客户？ | "3年内成为战略合作伙伴，年营收从500万增长到2000万" |
| **2. 经营维度** | 从哪些角度经营？ | "关系+业务+价值+口碑"四维经营 |
| **3. 关键人地图** | 谁是影响经营目标的关键人？ | "CEO（决策者）+ CIO（影响者）+ 业务总监（使用者）" |
| **4. 关系发展策略** | 如何从"陌生"到"同盟"？ | "第一阶段：建立信任（3个月）；第二阶段：深化合作（6个月）..." |
| **5. 业务渗透策略** | 如何从"单产品"到"多产品"？ | "第一年：CRM；第二年：营销云；第三年：PaaS平台" |
| **6. 价值呈现策略** | 如何量化价值？ | "帮助客户降低销售成本15%，年节省300万" |
| **7. 关键行动清单** | 具体要做什么？ | "Q1：CEO会晤；Q2：标杆案例打造；Q3：联合创新..." |
| **8. 资源需求** | 需要什么支持？ | "售前支持（20人天）+ 管理层介入（2次）+ 生态协同（1家）" |

---

### 第四步：设计管理机制

**智能提醒规则**（从人工到智能）：

```
规则1：拜访频次提醒
  ├─ S级客户：每周1次拜访（系统自动提醒）
  ├─ A级客户：每两周1次拜访
  ├─ B级客户：每月1次拜访
  └─ C级客户：每季度1次拜访

规则2：关键人变动提醒
  ├─ 监测：工商变更/离职/晋升
  ├─ 触发：关键人变动 → 自动提醒销售
  └─ 行动：7天内安排新关键人拜访

规则3：商机阶段停留超时提醒
  ├─ 监测：商机在当前阶段停留时间
  ├─ 触发：超过标准周期（如：方案阶段>30天）
  └─ 行动：自动提醒销售经理介入

规则4：客户满意度下滑提醒
  ├─ 监测：NPS评分/投诉记录
  ├─ 触发：NPS<6 或 投诉>2次/季度
  └─ 行动：自动触发"客户关怀"工作流
```

---

### 第五步：制定评估机制

**经营效果评估指标**：

| 维度 | 指标 | 目标值 | 评估周期 |
|:----|:------|:------|:--------|
| **关系指标** | 关键人覆盖率 | 80%+ | 季度 |
| **关系指标** | 拜访频次达标率 | 90%+ | 月度 |
| **关系指标** | 满意度评分（NPS） | 8+ | 半年 |
| **业务指标** | 营收完成率 | 100%+ | 月度 |
| **业务指标** | 回款完成率 | 95%+ | 月度 |
| **业务指标** | 商机数量 | 5+/年 | 季度 |
| **业务指标** | 赢单率 | 30%+ | 季度 |
| **价值指标** | 客户成功案例 | 1+/年 | 年度 |
| **价值指标** | 标杆打造 | 1+/年 | 年度 |
| **价值指标** | 口碑推荐（转介绍） | 2+/年 | 年度 |

**评估分析维度**：
- 纵向对比：本期 vs 上期
- 横向对比：本客户 vs 同层级客户
- 归因分析：达成/未达成的原因

## 01-A 客户全生命周期经营体系（v11.0新增）

> **来源**：张睿《CRM解决方案落地策略V5.2》
> **核心理念**：第一视角围绕客户经营价值链的关键阶段去定义价值落地的举措。

### 四阶段模型

```
潜在客户 → 意向客户 → 成交客户 → 流失客户
  MTL         LTC          MCR          ITR
```

| 阶段 | 核心任务 | 关键举措 | 产出 |
|:---:|---------|---------|------|
| **潜在客户** | ABM精准营销 | 目标客户定义、动态数据获取、ABM工具落地 | 列名客户库+客户画像 |
| **意向客户** | 项目驱动协作 | ☆评级体系、项目立项跟进、产销协同 | 商机档案+赢单策略 |
| **成交客户** | 忠诚度经营 | 忠诚度指标体系、分层经营策略、健康度模型 | 客户经营计划 |
| **流失客户** | 预警与挽回 | 四类风险信号监测、流失挽回五步法 | 流失挽回方案 |

### 阶段衔接的四大支柱

| 支柱 | 覆盖阶段 | 核心动作 |
|:---:|:-------:|---------|
| **MCR（客户关系管理）** | 全阶段 | 客户类型/阶段分群，针对性经营计划 |
| **经营六步法** | 意向+成交 | 市场抓机会→阶段定策略→PDCA闭环 |
| **LTC业务链** | 意向 | 还原流程，建立指标体系，控制关键点 |
| **S&OP业务链** | 成交 | 产销一体化，异构系统融合，统一业务中台 |

## 01-B 忠诚度管理体系（v11.0新增）

> **来源**：张睿《CRM解决方案落地策略V5.2》成交客户阶段

### 忠诚度核心指标权重

| 指标 | 权重 | 含义 |
|------|:---:|------|
| 客户购买总量中所占份额 | 30% | 占客户总采购的比例 |
| 顾客认知价值 | 20% | 客户对我们的价值感知 |
| 购买持续期 | 15% | 连续合作时间 |
| 与最大竞争对手相比所占份额 | 15% | 相对竞品的渗透率 |
| 客户信任 | 15% | 信任度评分 |
| 客户忠诚 | 15% | 忠诚行为指标 |
| 购买频率 | 10% | 年度采购次数 |
| 购买频率变化趋势 | 10% | 采购频率是上升还是下降 |
| 交叉销售 | 10% | 多产品/多部门覆盖 |
| 转换成本 | 15% | 客户切换到竞品的成本 |
| 产品被提及率 | 10% | 客户主动推荐频率 |

### 四层分级经营策略

| 忠诚度层级 | 经营策略 | 经营机制 |
|-----------|---------|---------|
| **高忠诚度** | 加强维护，个性化服务，增加交流互动，提供更优质产品和服务 | VIP客户管理制度，专属客户服务团队，定制化产品和服务 |
| **中忠诚度** | 加强关系维护，更多优惠和关怀，提高产品品质和性价比 | 客户关系管理制度，定期沟通互动，客户档案体系，跟踪需求反馈 |
| **低忠诚度** | 加强关系建立，个性化服务，刺激消费行为，增加忠诚度 | 客户挽留和恢复制度，个性化产品和服务，多种方式建立联系沟通 |
| **丢失客户** | 积极恢复，了解需求和原因，提供解决方案，恢复信任信心 | 客户流失和恢复管理制度，及时了解流失原因，反馈机制，跟踪纠正 |

### 忠诚度提升五步法

```
① 问卷设计 → ② 问卷采集（线下/在线/电话/邮件）
→ ③ 分析数据（汇总统计，数据挖掘，找出潜在问题）
→ ④ 提升忠诚度（改进产品品质/服务质量/客户体验）
→ ⑤ 持续跟踪（定期调查分析，保持需求敏感度）
```

## 01-C 流失预警与挽回体系（v11.0新增）

> **来源**：张睿《CRM解决方案落地策略V5.2》流失客户阶段

### 流失预警四类风险信号

| 风险类型 | 信号特征 | 监测方式 |
|:---:|------|------|
| **消费行为风险** | 购买频率↓、消费金额↓ | CRM交易数据监测 |
| **投诉退货风险** | 投诉/退货次数↑ | 客服工单统计 |
| **活跃度黏性风险** | 登录频次↓、使用时长↓、功能使用↓ | 系统埋点+健康度模型 |
| **竞品偏好风险** | 开始关注竞品、试用竞品 | Coach情报+生态信息 |

### 流失挽回五步法

```
① 及时了解客户的需求和反馈
② 通过多种渠道与客户建立联系
③ 提供个性化的服务和优惠
④ 建立客户关系管理体系
⑤ 针对潜在流失客户进行关注和营销
```

### 减少流失七大关键举措

1. 建立健全的客户服务体系和机制
2. 定期开展客户调研和满意度调查
3. 建立有效的客户信息管理系统
4. 持续推出具有竞争力的产品和服务
5. 加强客户关系管理和沟通
6. 优化营销策略和渠道
7. 及时响应和处理客户投诉和理赔问题

### 流失客户管理体系五步法

```
① 客户流失预测 → 分析客户信息+相关数据 → 预测可能流失客户
② 客户分析 → 对流失客户细致分析 → 找出流失原因
③ 解决方案制定 → 符合客户需求+公司利益的方案
④ 营销和沟通 → 积极互动沟通，提供优惠活动
⑤ 监测和改进 → 跟踪客户反馈和结果，持续改进
```

## 01-D 客户健康度模型（v11.0新增）

### 五步构建法

```
① 定义指标 → 定义客户健康度的指标体系
② 数据收集 → 收集相关数据（交易/行为/服务/满意度）
③ 数据分析 → 直观数据可视化，发掘规律和趋势
④ 模型建立 → 构建可洞察的客户健康度模型，支撑针对性策略
⑤ 模型更新 → 根据业务发展不断优化改进模型
```

### 健康度九宫格（源自YXAQ案例）

> 实时掌控客户的使用情况，通过健康度九宫格模型进行可视化监控

```
        高活跃度    中活跃度    低活跃度
高价值    🟢 核心    🟡 关注    🟠 预警
中价值    🟢 增长    🟡 维护    🟠 流失风险
低价值    🟡 培育    🟠 评估    🔴 放弃
```

### 三年经营目标示例

**案例：某装备制造企业（年营收5亿，CRM年付费100万）**

| 年份 | 经营目标 | 关键行动 | 营收目标 |
|:----|:--------|:--------|:--------|
| **第一年** | 建立信任（关系突破） | 1. CRM上线成功<br>2. 关键人关系建立<br>3. 首次满意度调研 | 100万（CRM） |
| **第二年** | 深化合作（价值呈现） | 1. 营销云增购<br>2. 标杆案例打造<br>3. CEO会晤（2次） | 200万（+营销云） |
| **第三年** | 战略伙伴（共生共赢） | 1. PaaS平台部署<br>2. 联合创新实验室<br>3. 生态协同（推荐2家客户） | 300万（+PaaS） |

**逐年营收/回款目标**：
- 第一年：营收100万，回款95万
- 第二年：营收200万，回款190万
- 第三年：营收300万，回款285万

## 05 质量标准

| 维度 | 标准 | 检查方法 |
